Evaluation of Section Properties

Exercises on Semi-Monocoque Beam Sections

Determine the stiffnesses \(EA\), \(EJ_{xx}\), \(EJ_{yy}\), \(GA_x^*\), \(GA_y^*\) and \(GJ\) for the beam sections reported in the following exercises. (The semi-monocoque scheme is used for solving the section)

Exercise 1: Rectangular Cross-Section with Four Corner Nodes

Given Data:

  • \(a = 600\, \text{mm}\)
  • \(b = 700\, \text{mm}\)
  • \(A = 300\, \text{mm}^2\)
  • \(t = 1.5\, \text{mm}\)

Solution

Principal Axes

The principal axes are available from the symmetry of the section.

Moments of Inertia

\[J_{xx} = 2A\left(\frac{a}{2}\right)^2 \cdot 2 = Aa^2\] \[J_{yy} = Ab^2\]

Bending Stiffnesses

\[EJ_{xx} = EAa^2\] \[EJ_{yy} = EAb^2\] \[(EA) = E\sum_{i=1}^{4} A_i = 4EA\]

Shear Stiffness \(GA_y^*\)

The evaluation of \(GA_y^*\) requires a previous analysis of the shear flows for an arbitrary force \(T_y\). As the section is open, it is assumed that the shear force is applied on the shear center.

First moments of area:

\[S_{x_1}' = -A \cdot \frac{a}{2}, \quad S_{x_2}' = -Aa, \quad S_{x_3}' = -A \cdot \frac{a}{2}\] \[J_{xx} = Aa^2\]

Shear flows:

\[q_1 = -\frac{T_y S_{x_1}'}{J_{xx}} = +\frac{T}{2a}\] \[q_2 = -\frac{T_y S_{x_2}'}{J_{xx}} = +\frac{T}{a}\] \[q_3 = -\frac{T_y S_{x_3}'}{J_{xx}} = +\frac{T}{2a}\]

Energy equivalence:

\[\frac{T^2}{GA_y^*} = \sum_i \frac{q_i^2 \ell_i}{Gt_i} \Rightarrow A_y^* = \frac{T^2}{\sum_i \frac{q_i^2 \ell_i}{t_i}}\] \[\sum_i \frac{q_i^2 \ell_i}{t_i} = \left[\left(\frac{T}{2a}\right)^2 b + \left(\frac{T}{a}\right)^2 a + \left(\frac{T}{2a}\right)^2 b\right] \frac{1}{t} = \frac{T^2}{t}\left(\frac{2b + 4a}{4a^2}\right)\] \[A_y^* = \frac{4a^2 t}{2b + 4a}\]
\[\boxed{GA_y^* = G\frac{4a^2 t}{2b + 4a}}\]

Shear Stiffness \(GA_x^*\)

\(T_x\) is applied on the shear center (the y position is available from the symmetry of the section).

The solution is readily found from the symmetry of the section with respect to the x-axis:

\[q_3 = q_1 = \frac{T}{2b}\]

Energy equivalence:

\[A_x^* = \frac{T^2}{\sum_i \frac{q_i^2 \ell_i}{t_i}} = \frac{t}{\frac{1}{4b} + \frac{1}{4b}} = 2bt\]
\[\boxed{GA_x^* = G \cdot 2bt}\]

Torsional Stiffness \(GJ\)

No torsional stiffness is associated with the semi-monocoque approximation of the section. Note that \(GJ = 0\) is not an intrinsic property of an open section, but is associated with the model considered for describing the section behaviour.

\[\boxed{GJ = 0}\]

Exercise 2: Rectangular Cross-Section with Six Stringers

Given Data:

  • \(a = 150\, \text{mm}\)
  • \(b = 200\, \text{mm}\)
  • \(t = 1\, \text{mm}\)
  • \(A = 200\, \text{mm}^2\)
  • \(E = 72\, \text{GPa}\)
  • \(\nu = 0.3\)

Note: \(A\) includes the contribution of the panels

Solution

Axial Stiffness

\[(EA) = E\sum_i A_i = 6EA = 86.40 \cdot 10^6\, \text{N}\]

Bending Stiffness - Principal Axes

Centroid position:

\[X_{CG} = 0 \quad \text{(from symmetry)}\] \[Y_{CG} = \frac{4Ab^2}{6A} = \frac{2}{3}b = 133.33\, \text{mm}\] \[J_{xx} = 4A\left(\frac{1}{3}b\right)^2 + 2A\left(\frac{2}{3}b\right)^2 = \left(\frac{4}{9} + \frac{8}{9}\right)Ab^2 = \frac{4}{3}Ab^2\] \[J_{yy} = 6A\left(\frac{a}{2}\right)^2 = \frac{3}{2}Aa^2\]
\[EJ_{xx} = \frac{4}{3}EAb^2 = 7.68 \cdot 10^{11}\, \text{Nmm}^2\] \[EJ_{yy} = \frac{3}{2}EAa^2 = 4.86 \cdot 10^{11}\, \text{Nmm}^2\]

Shear Stiffness \(GA_y^*\)

The x position of the shear center is available from the symmetry of the section. Even the shear flows are readily available from the symmetry of the section.

\[A_y^* = \frac{T_y^2}{\sum_i \frac{q_i^2 \ell_i}{t_i}} = \frac{T_y^2}{T_y^2\left(\frac{1}{4b^2} \cdot b + \frac{1}{4b^2} \cdot b\right)\frac{1}{t}} = 2bt\]

Note that \(A_y^* = 2bt\) is the total area of the vertical webs. As the shear flow is constant along the webs, \(A_y^* = \chi A_{webs}\) and \(\chi = 1\).

Shear factor:

\[\chi = \frac{A_y^*}{A_{TOT}} = \frac{2bt}{6A} = 0.33\]

Note that \(\chi << 1\), meaning a small fraction of the total area contributes to shear stiffness ⟹ the section is characterized by non-negligible shear deformability.

\[GA_y^* = \chi G A_{TOT} = 1.11 \cdot 10^4\, \text{N}\]

Shear Stiffness \(GA_x^*\)

The position of the shear center along the y-axis is not available from the symmetry of the section.

\(T_x = 1000\, \text{N}\) (arbitrary)

The shear center (y-position) is obtained by applying:

  • a. 3 shear flow equations
  • b. 1 equivalence with the internal moment
  • c. 1 compatibility condition (\(\theta_i' = 0\))

First moments of area:

\[S_{y_2}' = -Aa, \quad S_{y_3}' = 0, \quad S_{y_4}' = A \cdot \frac{a}{2}\]

Open section shear flows:

\[q_2' = -T_x \frac{S_{y_2}'}{J_{yy}} = 4.44\, \text{N/mm}\] \[q_3' = 0\] \[q_4' = -T_x \frac{S_{y_4}'}{J_{yy}} = -2.22\, \text{N/mm}\]

From equivalence and compatibility:

\[Y_{sc} = 85.16\, \text{mm}, \quad q^* = -0.80\, \text{N/mm}\]

Final shear flows:

\[q_1 = -0.80\, \text{N/mm}, \quad q_2 = 3.64\, \text{N/mm}\] \[q_3 = -0.80\, \text{N/mm}, \quad q_4 = -3.02\, \text{N/mm}\] \[A_x^* = \frac{T_x^2}{\frac{q_1^2 b}{t} + \frac{q_3^2 b}{t} + \frac{q_2^2 a}{t} + \frac{q_4^2 a}{2t}} = 341.59\, \text{mm}^2\] \[\chi = \frac{A_x^*}{A_{TOT}} = 0.28\]
\[GA_x^* = \chi G A_{TOT} = 9.46 \cdot 10^6\, \text{N}\]

Torsional Stiffness \(GJ\)

The torsional constant \(J\) is obtained from:

\[J = \frac{4\Omega^2}{\oint_\Gamma \frac{1}{t(s)} d\ell}\]

Where:

\[\oint_\Gamma \frac{1}{t(s)} = \frac{1}{t}(2b + a) + \frac{1}{2t}a = \frac{1}{t}\left(\frac{4b + 3a}{2}\right)\] \[\Omega^2 = (ab)^2\] \[J = \frac{8a^2 b^2 t}{4b + 3a} = 5.76 \cdot 10^6\, \text{mm}^4\]
\[GJ = 1.60 \cdot 10^{11}\, \text{Nmm}^2\]

Exercise 3: Circular Cross-Section

Given Data:

  • \(R = 100\, \text{mm}\)
  • \(A = 500\, \text{mm}^2\)
  • \(t = 1.2\, \text{mm}\)
  • \(E = 72\, \text{GPa}\)
  • \(\nu = 0.3\)

Solution

Axial Stiffness

\[(EA) = E\sum_i A_i = 4EA = 1.44 \cdot 10^8\, \text{N}\]

Bending Stiffness

Thanks to the double symmetry of the section, the principal axes are readily available.

\[J_{xx} = 2AR^2\] \[J_{yy} = J_{xx}\]
\[EJ_{xx} = EJ_{yy} = 1.0 \cdot 10^7\, \text{Nmm}^2\]

Shear Stiffness \(GA_x^*\) and \(GA_y^*\)

The symmetry of the section is such that \(GA_x^* = GA_y^*\). The position of the shear center is available from the double symmetry of the section, and coincides with the centroid.

The shear flows are available by exploiting the symmetry:

\[q_i = \frac{T_y}{4R}\] \[A_y^* = \frac{T_y^2}{\frac{T_y^2 \cdot 2\pi R}{16R^2 t}} = \frac{8Rt}{\pi} = 305.58\, \text{mm}^2\] \[\chi_y = \frac{A_y^*}{A_{TOT}} = 0.15\]
\[GA_y^* = \chi_y G A_{TOT} = 8.46 \cdot 10^6\, \text{N}\] \[GA_x^* = GA_y^*\]

Exercise 4: Rectangular Frame with Two Cells

Given Data:

  • \(A = 500\, \text{mm}^2\)
  • \(a = 100\, \text{mm}\)
  • \(b = 300\, \text{mm}\)
  • \(c = 200\, \text{mm}\)
  • \(t = 1\, \text{mm}\)
  • \(E = 72\, \text{GPa}\)
  • \(\nu = 0.3\)

Solution

Axial Stiffness

\[(EA) = E\sum_i A_i = 6EA\]

Bending Stiffness - Principal Axes

\[X_{CG} = \frac{2A(a+b) + 2A(a)}{6A} = \frac{2a + b}{3} = 166.67\, \text{mm}\] \[Y_{CG} = 0\]

Dimensions:

\[d = X_{CG} = 166.67\, \text{mm}\] \[e = X_{CG} - a = 66.67\, \text{mm}\] \[f = a + b - d = 233.33\, \text{mm}\] \[J_{xx} = 6A\left(\frac{c}{2}\right)^2 = \frac{3}{2}Ac^2\] \[J_{yy} = 2Af^2 + 2Ae^2 + 2Ad^2\]
\[EJ_{xx} = \frac{3}{2}EAc^2 = 2.16 \cdot 10^{12}\, \text{Nmm}^2\] \[EJ_{yy} = 2EA\left(f^2 + e^2 + d^2\right) = 6.24 \cdot 10^{12}\, \text{Nmm}^2\]

Shear Stiffness \(GA_y^*\)

The position of the shear center has to be evaluated. The equations to be applied are:

  • a. 5 shear flow equations
  • b. 1 equivalence with internal moment
  • c. 1 compatibility condition (\(\theta'_{c_1} = \theta'_{c_2}\))
  • d. 1 compatibility condition (\(\theta'_{c_1} = 0\))

First moments of area:

\[S_{x_1}' = \frac{Ac}{2}, \quad S_{x_2}' = Ac, \quad S_{x_3}' = \frac{3}{2}Ac\] \[S_{x_4}' = Ac, \quad S_{x_5}' = \frac{Ac}{2}\]

Open section shear flows:

\[q_1' = -1.67\, \text{N/mm}, \quad q_2' = -3.33\, \text{N/mm}, \quad q_3' = -5.00\, \text{N/mm}\] \[q_4' = -3.33\, \text{N/mm}, \quad q_5' = -1.67\, \text{N/mm}\]

Solving the system of equations:

\[q_1^* = 1.6666\, \text{N/mm}, \quad q_2^* = 3.3332\, \text{N/mm}\] \[X_{sc} = 166.68\, \text{mm}\]

Final shear flows:

\[q_1 = q_2 = q_4 = q_5 = 0\] \[q_3 = -1.67\, \text{N/mm}\] \[q_6 = q_7 = 1.67\, \text{N/mm}\] \[A_y^* = 600\, \text{mm}^2, \quad \chi_y = 0.2\]
\[GA_y^* = \chi_y G A_{TOT} = 1.66 \cdot 10^7\, \text{N}\]

Shear Stiffness \(GA_x^*\)

The y-position of the shear center is available from the symmetry of the section.

First moments of area:

\[S_{y_1}' = -Ad, \quad S_{y_2}' = -Ae + S_{y_1}', \quad S_{y_3}' = Af + S_{y_2}'\]

Open section shear flows:

\[q_1' = 0.96\, \text{N/mm}, \quad q_2' = 1.35\, \text{N/mm}, \quad q_3' = 0\] \[q_4' = -1.35\, \text{N/mm}, \quad q_5' = -0.96\, \text{N/mm}\]

Solving gives \(q_1^* = q_2^* = 0\), and:

\[q_1 = 0.96, \quad q_2 = 1.35, \quad q_3 = q_6 = q_7 = 0\] \[q_4 = -1.35, \quad q_5 = -0.96\] \[A_x^* = 786.04\, \text{mm}^2, \quad \chi_x = 0.26\]
\[GA_x^* = \chi_x G A_{TOT} = 2.17 \cdot 10^7\, \text{N}\]

Torsional Stiffness \(GJ\)

The torsional stiffness cannot be evaluated using the formula for single-cell sections. We use:

\[GJ = \frac{M_z}{\theta'}\]

Solving the shear flow equations with compatibility \(\theta_1' = \theta_2'\):

\[q_1^* = 5.26\, \text{N/mm}, \quad q_2^* = 6.58\, \text{N/mm}\] \[q_1 = q_5 = q_6 = 5.26\, \text{N/mm}\] \[q_2 = q_3 = q_4 = 6.58\, \text{N/mm}\] \[q_7 = 1.32\, \text{N/mm}\] \[\theta_1' = \frac{1}{2G\Omega_{c_1} t}\left(q_1 \cdot 2a + q_6 \cdot c - q_7 \cdot c\right) = 1.663 \cdot 10^{-6}\, \text{rad}\] \[J = \frac{M_z}{G\theta'} = 2.17 \cdot 10^7\, \text{mm}^4\]
\[GJ = 6.04 \cdot 10^{11}\, \text{N} \cdot \text{mm}^2\]

Exercise 5: Semicircular Cross-Section

Given Data:

  • \(R = 100\, \text{mm}\)
  • \(A = 500\, \text{mm}^2\)
  • \(t = 1\, \text{mm}\)

Solution

Axial Stiffness

\[(EA) = E\sum_i A_i = 5EA = 1.8 \cdot 10^8\, \text{N}\]

Bending Stiffness

Centroid position:

\[Y_{CG} = 0\] \[X_{CG} = \frac{AR + AR/2}{5A} = \frac{3}{10}R = 30\, \text{mm}\]

Dimensions:

\[d = \frac{3}{10}R = 30\, \text{mm}\] \[e = R/2 - d = 20\, \text{mm}\] \[f = R/2 + e = 70\, \text{mm}\] \[J_{xx} = 2AR^2\] \[J_{yy} = 3Ad^2 + Ae^2 + Af^2\]
\[EJ_{xx} = 7.20 \cdot 10^{11}\, \text{Nmm}^2\] \[EJ_{yy} = 2.88 \cdot 10^{11}\, \text{Nmm}^2\]

Shear Stiffness \(GA_y^*\)

The x-position of the shear center is not known a priori and has to be determined.

Applying shear flow equations, equivalence, and compatibility conditions:

\[q_1 = q_2 = 3.06\, \text{N/mm}\] \[q_3 = q_4 = -1.95\, \text{N/mm}\] \[q_5 = q_6 = 0\] \[X_{sc} = 61.10\, \text{mm}\] \[A_y^* = 326\, \text{mm}^2, \quad \chi_y = 0.13\]
\[GA_y^* = \chi G A_{TOT} = 9.03 \cdot 10^6\, \text{N}\]

Shear Stiffness \(GA_x^*\)

The y position of the shear center is known from the symmetry.

\[q_1 = 0.74\, \text{N/mm}, \quad q_2 = -0.74\, \text{N/mm}\] \[q_3 = 3.01\, \text{N/mm}, \quad q_4 = -3.01\, \text{N/mm}\] \[q_5 = -2.73\, \text{N/mm}, \quad q_6 = -5.23\, \text{N/mm}\] \[A_x^* = 213\, \text{mm}^2, \quad \chi_x = 0.09\]
\[GA_x^* = 5.90 \cdot 10^6\, \text{N}\]

Torsional Stiffness \(GJ\)

The section is composed of two cells, so an arbitrary torsional moment is applied:

\[\theta_1' = \theta_2' = 1.88 \cdot 10^{-5}\, \text{rad}\] \[J = \frac{M_z}{G\theta_1'} = 1.92 \cdot 10^6\, \text{mm}^4\]
\[GJ = 5.32 \cdot 10^{10}\, \text{Nmm}^2\]

Exercise 6: Rectangular Frame with Two Cells (Panel Configuration)

Given Data:

  • \(A = 400\, \text{mm}^2\)
  • \(a = 110\, \text{mm}\)
  • \(b = 70\, \text{mm}\)
  • \(t = 1.3\, \text{mm}\)
  • \(E = 72\, \text{GPa}\)
  • \(\nu = 0.3\)

Note: \(A\) denotes the lumped area (the contribution of the panels is already included)

Solution

Axial Stiffness

\[(EA) = 6EA = 1.73 \cdot 10^8\, \text{N}\]

Bending Stiffness

The principal axes are available from the symmetry:

\[J_{xx} = 6A\left(\frac{b}{2}\right)^2 = \frac{3}{2}Ab^2\] \[J_{yy} = 4Aa^2\]
\[EJ_{xx} = 2.12 \cdot 10^{11}\, \text{Nmm}^2\] \[EJ_{yy} = 1.40 \cdot 10^{12}\, \text{Nmm}^2\]

Shear Stiffness \(GA_x^*\)

The y-position of the shear center is available from the symmetry.

The solution can be found easily by noting that:

  1. A symmetry with respect to x exists, so: \(q_3 = q_6 = 0\), \(q_1 = q_5\) and \(q_4 = q_2\)
  2. The stringers 2 and 5 are located on the neutral axis ⟹ \(q_1 = q_2\) and \(q_4 = q_5\)

The shear flows are constant along the y-direction: \(q = \frac{T_x}{4a}\)

\[A_x^* = A_{webs} = 4at\]
\[GA_x^* = 1.58 \cdot 10^7\, \text{N}\]

Shear Stiffness \(GA_y^*\)

The x-position of the shear center cannot be determined from symmetry considerations (panel 3 has thickness \(2t\), while the others \(t\)).

Applying the full set of equations:

\[A_y^* = 33.5\, \text{mm}^2\]

Note: \(A_y^*\) is smaller than the total area of the vertical webs. When the shear flows are constant and equal on the webs, then \(A^* = A_{web}\); otherwise \(A^* < A_{web}\).

\[GA_y^* = 9.28 \cdot 10^6\, \text{N}\]

Torsional Stiffness \(GJ\)

This is a multicell configuration. Applying \(M_z = 1 \cdot 10^6\, \text{Nmm}\):

\[q_1 = q_6 = 31.09\, \text{N/mm}\] \[q_2 = q_4 = 33.84\, \text{N/mm}\] \[q_7 = 2.75\, \text{N/mm}\] \[\theta_1' = \theta_2' = 1.59 \cdot 10^{-5}\, \text{rad}\]
\[GJ = \frac{M_z}{\theta'} = 6.28 \cdot 10^{10}\, \text{N} \cdot \text{mm}^2\]

Exercise 7: Cantilever Beam with Rectangular Cross-Section

Given Data:

  • \(\ell = 1800\, \text{mm}\)
  • \(a = 150\, \text{mm}\)
  • \(b = 250\, \text{mm}\)
  • \(t = 1.5\, \text{mm}\)
  • \(E = 72\, \text{GPa}\)
  • \(\nu = 0.3\)
  • \(P = 2000\, \text{N}\)
  • \(A = 300\, \text{mm}^2\)

Evaluate the displacement \(S\) (vertical) due to the applied force \(P\). Quantify the contribution due to the shear deformability.

Solution

Section Properties

Due to the symmetry, the centroid and shear center are coincident:

\[J_{xx} = Aa^2\] \[EJ_{xx} = 4.86 \cdot 10^{11}\, \text{Nmm}^2\]

Shear Stiffness \(GA_y^*\)

Since the shear center position is known, the shear flow is \(q = \frac{\bar{T}}{2a}\) constant on vertical webs:

\[A_y^* = 2at\] \[GA_y^* = 1.25 \cdot 10^7\, \text{N}\]

Torsional Stiffness

\[J = \frac{4\Omega^2}{\oint_\Gamma \frac{1}{t(s)} ds} = \frac{2ta^2b^2}{a + b}\] \[GJ = 2.92 \cdot 10^{11}\, \text{Nmm}^2\]

PCVW Application

Real system:

\[T_y = P, \quad M_x = Pz, \quad M_z = -\frac{Pb}{2}\]

Dummy system:

\[\delta T_y = 1, \quad \delta M_x = z, \quad \delta M_z = \frac{b}{2}\]

The PCVW:

\[\int_0^\ell \left(\delta M_x \frac{M_x}{EJ} + \delta T_y \frac{T_y}{GA_y^*} + \delta M_z \frac{M_z}{GJ}\right) dz = 1 \cdot S\] \[\int_0^\ell \left(\frac{Pz^2}{EJ} + \frac{P}{GA_y^*} - P\frac{b^2}{4GJ}\right) dz = S\]
\[S = P\left(\frac{\ell^3}{3EJ} + \frac{\ell}{GA_y^*} - \frac{b^2 \ell}{4GJ}\right) = 12.14\, \text{mm}\]

Contribution Analysis

\[S^{shear} = \frac{\ell}{GA_y^*} P = 0.43\, \text{mm}\] \[S^{bend} = P\frac{\ell^3}{3EJ} = 12\, \text{mm}\]

Exercise 8: Two-Segment Beam Structure

Given Data:

  • \(a = 100\, \text{mm}\)
  • \(b = 300\, \text{mm}\)
  • \(c = 200\, \text{mm}\)
  • \(t = 1\, \text{mm}\)
  • \(A = 500\, \text{mm}^2\)
  • \(\ell = 900\, \text{mm}\)
  • \(P = 1000\, \text{N}\)

Evaluate the displacement \(S\) and assess the contribution of shear deformability in relation to the bending one.

Solution

Section Properties (from Exercise 4)

\[EJ_{xx} = 2.16 \cdot 10^{12}\, \text{Nmm}^2\] \[GA_y^* = 1.66 \cdot 10^7\, \text{N}\] \[GJ = 6.01 \cdot 10^{11}\, \text{Nmm}^2\] \[f = 233.33\, \text{mm}\]

Load Transfer to Shear Center

The load \(P\) at the stringer creates a force \(P\) and moment \(Pf\) at the shear center.

Real System Internal Forces

Beam 1:

\[T_y^{(1)} = P, \quad M_x^{(1)} = Pz_1, \quad M_z^{(1)} = -Pf\]

Beam 2:

\[T_y^{(2)} = 2P, \quad M_x^{(2)} = P\ell + 2Pz_2, \quad M_z^{(2)} = -2Pf\]

Dummy System

Beam 1:

\[\delta T_y^{(1)} = 1, \quad \delta M_x^{(1)} = z_1, \quad \delta M_z^{(1)} = -f\]

Beam 2:

\[\delta T_y^{(2)} = 1, \quad \delta M_x^{(2)} = \ell + z_2, \quad \delta M_z^{(2)} = -f\]

PCVW Integration

\[\int_0^\ell \left(\frac{T_y^{(1)} \delta T_y^{(1)}}{GA_y^*} + \frac{M_x^{(1)} \delta M_x^{(1)}}{EJ_{xx}} + \frac{M_z^{(1)} \delta M_z^{(1)}}{GJ}\right) dz_1 +\] \[\int_0^\ell \left(\frac{T_y^{(2)} \delta T_y^{(2)}}{GA_y^*} + \frac{M_x^{(2)} \delta M_x^{(2)}}{EJ_{xx}} + \frac{M_z^{(2)} \delta M_z^{(2)}}{GJ}\right) dz_2 = S\]

After integration:

\[P\left(\frac{7}{2}\frac{\ell^3}{EJ} + \frac{3\ell}{GA_y^*} + \frac{3f^2\ell}{GJ}\right) = S\]
\[S = 1.59\, \text{mm}\] \[S_{shear} = \frac{3\ell}{GA_y^*} P = 0.16\, \text{mm}\] \[S_{bending} = \frac{7}{2}\frac{\ell^3}{EJ} P = 1.18\, \text{mm}\]

Exercise 9: Beam with Spring Supports (Statically Indeterminate)

Given Data:

  • \(a = 100\, \text{mm}\)
  • \(b = 300\, \text{mm}\)
  • \(c = 200\, \text{mm}\)
  • \(t = 1\, \text{mm}\)
  • \(A = 500\, \text{mm}^2\)
  • \(\ell = 900\, \text{mm}\)
  • \(P = 5000\, \text{N}\)
  • \(k = 10^6\, \text{N/mm}\)
  • \(E = 72\, \text{GPa}\)
  • \(\nu = 0.3\)

Determine the reaction forces in correspondence of the linear springs.

Solution

Section Properties

\[EJ_{xx} = 2.16 \cdot 10^{12}\, \text{Nmm}^2\] \[GA_y^* = 1.66 \cdot 10^7\, \text{N}\] \[GJ = 6.01 \cdot 10^{11}\, \text{Nmm}^2\]

The shear center coincides with the centroid.

Problem Setup

The beam is two times statically indeterminate. The problem is solved by applying two times the PCVW.

Internal actions referred to the shear center:

At spring support: Force \((X+Y)\) upward, Moment \((Xd - Yf)\)

At load location: Force \(2P\) upward, Moment \(P(d-f)\)

Real System Internal Forces

Beam 1:

\[T_y^{(1)} = -(X+Y)\] \[M_x^{(1)} = -(X+Y)z_1\] \[M_z^{(1)} = -(Xd - Yf)\]

Beam 2:

\[T_y^{(2)} = -(2P + X + Y)\] \[M_x^{(2)} = -(2P + X + Y)z_2 - (X+Y)\ell\] \[M_z^{(2)} = -(Xd - Yf) - P(d-f)\]

Dummy System 1

Beam 1:

\[^1\delta T_y^{(1)} = -1, \quad ^1\delta M_x^{(1)} = -z_1, \quad ^1\delta M_z^{(1)} = -d\]

Beam 2:

\[^1\delta T_y^{(2)} = -1, \quad ^1\delta M_x^{(2)} = -(\ell + z_2), \quad ^1\delta M_z^{(2)} = -d\]

Dummy System 2

Beam 1:

\[^2\delta T_y^{(1)} = -1, \quad ^2\delta M_x^{(1)} = -z_1, \quad ^2\delta M_z^{(1)} = f\]

Beam 2:

\[^2\delta T_y^{(2)} = -1, \quad ^2\delta M_x^{(2)} = -\ell - z_2, \quad ^2\delta M_z^{(2)} = f\]

Exercise 10: Wing Structure with Pressure Loading

Given Data:

  • \(a = 100\, \text{mm}\)
  • \(b = 150\, \text{mm}\)
  • \(t = 1\, \text{mm}\)
  • \(A = 300\, \text{mm}^2\)
  • \(\ell = 400\, \text{mm}\)
  • \(E = 72\, \text{GPa}\)
  • \(\nu = 0.3\)
  • \(p_o = 0.02\, \text{N/mm}^2\)

Evaluate the displacement \(S\) due to the pressure load \(p\) (constant along the chordwise direction and linear along the spanwise direction).

Solution

Section Properties

\[J_{xx} = 4A\left(\frac{a}{2}\right)^2 = Aa^2\] \[EJ_{xx} = 2.16 \cdot 10^{11}\, \text{Nmm}^2\] \[A_y^* = 2at = 200\, \text{mm}^2\] \[GA_y^* = 5.54 \cdot 10^6\, \text{N}\] \[J = \frac{2a^2b^2t}{a+b}\] \[GJ = 4.98 \cdot 10^{10}\, \text{N}\]

Structure Analysis

The structure is 1 time statically indeterminate. The beam is transversally stiffened with 4 ribs (R1, R2, R3, R4) which are responsible for the load introduction.

The pressure varies linearly along the span:

\[p(z) = \frac{3p_o}{3\ell}z = \frac{p_o}{\ell}z\]

Integrated loads at each rib:

\[q_1 = \int_0^{\ell/2} p(z)\, dz = \frac{p_o \ell}{8}\] \[q_2 = \int_{\ell/2}^{3\ell/2} p(z)\, dz = p_o \ell\] \[q_3 = \int_{3\ell/2}^{5\ell/2} p(z)\, dz = 2p_o \ell\] \[q_4 = \int_{5\ell/2}^{3\ell} p(z)\, dz = \frac{11}{8} p_o \ell\]

Evaluation of Reaction Force

Shear forces applied at the shear center:

\[F_1 = q_1 b = 150\, \text{N}\] \[F_2 = q_2 b = 1200\, \text{N}\] \[F_3 = q_3 b = 2400\, \text{N}\]

Real System Internal Forces

Beam #1:

\[T_y^{(1)} = -(X + F_1)\] \[M_x^{(1)} = -(X + F_1)z_1\] \[M_z^{(1)} = -X\frac{b}{2}\]

Beam #2:

\[T_y^{(2)} = -(X + F_1 + F_2)\] \[M_x^{(2)} = -(F_1 + X)\ell - (X + F_1 + F_2)z_2\] \[M_z^{(2)} = -X\frac{b}{2}\]

Beam #3:

\[T_y^{(3)} = -(X + F_1 + F_2 + F_3)\] \[M_x^{(3)} = -(X + F_1 + F_2 + F_3)z_3 - (F_1 + X)2\ell - F_2\ell\] \[M_z^{(3)} = -X\frac{b}{2}\]

Dummy System for X

Beam #1:

\[\delta T_y^{(1)} = -1, \quad \delta M_x^{(1)} = -z_1, \quad \delta M_z^{(1)} = -\frac{b}{2}\]

Beam #2:

\[\delta T_y^{(2)} = -1, \quad \delta M_x^{(2)} = -(\ell + z_2), \quad \delta M_z^{(2)} = -\frac{b}{2}\]

Beam #3:

\[\delta T_y^{(3)} = -1, \quad \delta M_x^{(3)} = -(2\ell + z_3), \quad \delta M_z^{(3)} = -\frac{b}{2}\]

Compatibility Condition

\[\sum_{i=1}^{3} \int_0^\ell \left(\delta T_y^{(i)} \frac{T_y}{GA_y^*} + \delta M_x^{(i)} \frac{M_x}{EJ_{xx}} + \delta M_z^{(i)} \frac{M_z}{GJ}\right) dz_i = 0\]

Which leads to:

\[\left(\frac{13}{2}\frac{\ell^3}{EJ} + \frac{3\ell}{GA} + \frac{3b^2\ell}{4GJ}\right)X = -\frac{\ell}{GA_y^*}\left(3F_1 + 2F_2 + F_3\right) - \frac{\ell^3}{EJ}\left(\frac{13}{2}F_1 + \frac{13}{6}F_2\right)\] \[X = -631\, \text{N}\]

Evaluation of Displacement S

The dummy system differs from the previous one only for the sign of \(\delta M_z^{(i)}\) (now \(+\frac{b}{2}\)).

The compatibility condition reads:

\[\sum_{i=1}^{3} \int_0^\ell \left(\delta T_y^{(i)} \frac{T_y^{(i)}}{GA_y^*} + \delta M_x^{(i)} \frac{M_x^{(i)}}{EJ_{xx}} + \delta M_z^{(i)} \frac{M_z^{(i)}}{GJ}\right) dz_i = S\]
\[S = \frac{\ell}{GA_y^*}\left(3F_1 + 3X + 2F_2 + F_3\right) + \frac{\ell^3}{EJ}\left(\frac{13}{2}(F_1 + X) + \frac{13}{6}F_2\right) - \frac{3}{4}\frac{b^2\ell X}{GJ}\] \[\boxed{S = 0.17\, \text{mm}}\]
