Structural Mechanics: Ritz and Galerkin Methods

Exercise Solutions — Beam Bending, Torsion, and Vibration Analysis

Exercise 1: Cantilever Beam with Distributed Load

Compare the mid-displacement obtained from:

  • The exact solution
  • The Ritz solution using 1, 2, and 3 trigonometric functions
  • The Galerkin solution using 1 trigonometric function

Solution

1. Exact Solution

The governing equation and boundary conditions are:

\[\begin{cases} EJ \, W_{,xxxx} = q \\ W(0) = 0 \\ W_{,x}(0) = 0 \\ W(\ell) = 0 \\ W_{,x}(\ell) = 0 \end{cases} \leftarrow \text{essential conditions}\]

The solution is given by \(W = W^H + W^P\) where:

\[W^P = \frac{1}{24} \frac{q}{EJ} x^4\] \[W^H = A_0 + A_1 x + A_2 x^2 + A_3 x^3\]

Imposing the boundary conditions:

\[W(0) = 0 \quad \Rightarrow \quad A_0 = 0\] \[W_{,x}(0) = 0 \quad \Rightarrow \quad A_1 = 0\] \[W(\ell) = 0 \quad \Rightarrow \quad A_2 + A_3 \ell + \frac{q}{24EJ} \ell^2 = 0\] \[W_{,x}(\ell) = 0 \quad \Rightarrow \quad 2A_2 + 3A_3 \ell + \frac{q}{6EJ} \ell^2 = 0\]

Which leads to:

\[A_3 = -\frac{q}{12EJ} \ell \quad ; \quad A_2 = \frac{q}{24EJ} \ell^2\]

The exact displacement field is:

\[W = \frac{q\ell^4}{24EJ} \left[ \left(\frac{x}{\ell}\right)^2 - 2\left(\frac{x}{\ell}\right)^3 + \left(\frac{x}{\ell}\right)^4 \right]\]

Mid-span displacement:

\[W(\ell/2) = \frac{q\ell^4}{384EJ}\]

2. Ritz Solution — 1 Term

The trial function is taken as:

\[\phi_1 = 1 - \cos \frac{2\pi x}{\ell}\]

and \(W = C_1 \phi_1\)

Note that the trial function satisfies the 4 essential conditions:

\[W(0) = W_{,x}(0) = W(\ell) = W_{,x}(\ell) = 0\]

The stiffness matrix and the load vectors are:

\[K_{ij} = \int_0^\ell \phi_{i,xx} \, EJ \, \phi_{j,xx} \, dx\] \[f_i = \int_0^\ell \phi_i \, q \, dx\]

In this case:

\[K_{11} = \frac{8\pi^4 EJ}{\ell^3} \qquad f_1 = q\ell\]

The unknown amplitude is:

\[C_1 = \frac{f_1}{K_{11}} = \frac{q\ell^4}{8\pi^4 EJ}\]

The displacement field is:

\[W = \frac{q\ell^4}{8\pi^4 EJ} \left(1 - \cos \frac{2\pi x}{\ell}\right)\]

Mid-span displacement:

\[W(\ell/2) = \frac{q\ell^4}{4EJ\pi^4}\]

3. Ritz Solution — 2 Terms

Trial functions:

\[\phi_1 = 1 - \cos \frac{2\pi x}{\ell} \qquad \phi_2 = 1 - \cos \frac{4\pi x}{\ell}\]

The stiffness matrix reads:

\[K_{11} = \frac{8\pi^4 EJ}{\ell^3} \quad \text{(already available)}\] \[K_{12} = K_{21} = \int_0^\ell \phi_{1,xx} \, EJ \, \phi_{2,xx} \, dx = 0\] \[K_{22} = \int_0^\ell \phi_{2,xx} \, EJ \, \phi_{2,xx} \, dx = \frac{128\pi^4 EJ}{\ell^3}\]

The load vector is:

\[f_1 = q\ell \qquad f_2 = q\ell\]

The system to be solved is:

\[\begin{bmatrix} K_{11} & 0 \\ 0 & K_{22} \end{bmatrix} \begin{Bmatrix} C_1 \\ C_2 \end{Bmatrix} = \begin{Bmatrix} f_1 \\ f_2 \end{Bmatrix}\]

Which leads to:

\[C_1 = \frac{q\ell^4}{8EJ\pi^4} \qquad C_2 = \frac{q\ell^4}{128EJ\pi^4}\]

The displacement at mid-span is still:

\[W(\ell/2) = \frac{q\ell^4}{4EJ\pi^4}\]
Note: This is not surprising as \(\phi_2(\ell/2) = 0\)

4. Ritz Solution — 3 Terms

Trial functions:

\[\phi_1 = 1 - \cos \frac{2\pi x}{\ell} \; ; \; \phi_2 = 1 - \cos \frac{4\pi x}{\ell} \; ; \; \phi_3 = 1 - \cos \frac{6\pi x}{\ell}\]

Due to the orthogonality of the functions, the only terms to be computed are:

\[K_{33} = \int_0^\ell \phi_{3,xx} \, EJ \, \phi_{3,xx} \, dx = \frac{648 \, EJ \, \pi^4}{\ell^3}\] \[f_3 = \int_0^\ell \phi_3 \, q \, dx = q\ell\]

The final system is:

\[\begin{bmatrix} K_{11} & & \\ & K_{22} & \\ & & K_{33} \end{bmatrix} \begin{Bmatrix} C_1 \\ C_2 \\ C_3 \end{Bmatrix} = \begin{Bmatrix} f_1 \\ f_2 \\ f_3 \end{Bmatrix}\]

Leading to:

\[C_1 = \frac{q\ell^4}{8EJ\pi^4} \qquad C_2 = \frac{q\ell^4}{128EJ\pi^4} \qquad C_3 = \frac{q\ell^4}{648EJ\pi^4}\]

The mid-span displacement is then:

\[W(\ell/2) = \frac{41}{162} \frac{q\ell^4}{EJ\pi^4}\]

5. Galerkin Solution — 1 Term

The governing equation reads: \(EJ \, W_{,xxxx} = q\), so:

\[\int_0^\ell \phi_1 \, EJ \, \phi_{1,xxxx} \, dx \, C_1 = \int_0^\ell \phi_1 \, q \, dx\]

which is:

\[\frac{8EJ\pi^4}{\ell^3} C_1 = q\ell\] \[\Rightarrow \quad C_1 = \frac{q\ell^4}{8EJ\pi^4}\]
Note: Same result obtained with Ritz method.

Exercise 2: Torsion Problem with Distributed Moment

Obtain the exact displacement field and compare it with the 1-term solution obtained using Ritz with polynomial functions.

Solution

Exact Solution

Starting from the principle of virtual work:

\[\delta\Pi = \int_0^\ell \delta\theta_{,x} \, GJ \, \theta_{,x} \, dx - \int_0^\ell \delta\theta \, m_t \, dx\] \[= -\int_0^\ell \delta\theta \, GJ \, \theta_{,xx} \, dx + \delta\theta \, GJ \, \theta_{,x} \Big|_0^\ell - \int_0^\ell \delta\theta \, m_t \, dx\]

The governing equations and boundary conditions are:

\[\begin{cases} GJ\theta_{,xx} + m_t = 0 \\ GJ\theta_{,x}(0) = 0 & \text{or} & \delta\theta(0) = 0 \\ GJ\theta_{,x}(\ell) = 0 & \text{or} & \delta\theta(\ell) = 0 \end{cases}\]

The problem is then:

\[\begin{cases} \theta_{,xx} = -\frac{m_t}{GJ} \\ \theta(0) = 0 \\ \theta(\ell) = \bar{\theta} \end{cases}\]

The solution is found as \(\theta = \theta^H + \theta^P\) with:

\[\theta^P = -\frac{1}{2} \frac{m_t}{GJ} x^2 \qquad \theta^H = A_0 + A_1 x\]

Imposing the boundary conditions:

\[\theta(0) = 0 \quad \Rightarrow \quad A_0 = 0\] \[\theta(\ell) = \bar{\theta} \quad \Rightarrow \quad A_1 \ell - \frac{1}{2} \frac{m_t}{GJ} \ell^2 = \bar{\theta} \quad \Rightarrow \quad A_1 = \frac{\bar{\theta}}{\ell} + \frac{1}{2} \frac{m_t}{GJ} \ell\]

The exact displacement field is:

\[\theta(x) = \frac{1}{2} \frac{m_t \ell^2}{GJ} \left( \frac{x}{\ell} - \frac{x^2}{\ell^2} \right) + \frac{\bar{\theta}}{\ell} x\]

Ritz Solution — 1 Term

In this case the problem is characterized by two essential conditions. However, one of the two conditions is non-homogeneous.

The expansion is then taken as:

\[\theta = \sum_{i=1}^{N} C_i \phi_i + \phi_0\]

where:

  • \(\phi_i\) → satisfy the homogeneous part of the essential conditions
  • \(\phi_0\) → satisfies the non-homogeneous essential boundary conditions

More specifically:

\[\phi_i(0) = 0 \quad \text{and} \quad \phi_0(0) = 0\] \[\phi_i(\ell) = 0 \quad \text{and} \quad \phi_0(\ell) = \bar{\theta}\]

The simplest choice for \(\phi_0\) is a linear polynomial:

\[\phi_0(x) = \frac{\bar{\theta}}{\ell} x\]

The functions \(\phi_i\) should be taken as second order polynomial, so that the two conditions can be imposed:

\[\theta^H = a_0 + a_1 x + a_2 x^2\] \[\theta(0) = 0 \quad \Rightarrow \quad a_0 = 0\] \[\theta(\ell) = 0 \quad \Rightarrow \quad a_1 = -a_2 \ell\] \[\Rightarrow \quad \theta^H = a_2 (x^2 - \ell x)\]

The Ritz functions are then taken as:

\[\theta = C_1 (x^2 - \ell x) + \frac{\bar{\theta}}{\ell} x\]

Due to the fact that the exact solution is quadratic, the assumed Ritz expansion will lead to the exact solution. This is easily verified:

\[\int_0^\ell \delta\theta_{,x} \, GJ \, \theta_{,x} \, dx - \int_0^\ell \delta\theta \, m_t \, dx = 0\]

Since \(\theta = C_1 \phi_1 + \phi_0\) and \(\delta\theta = \delta C_1 \phi_1\):

\[\delta C_1 \underbrace{\int_0^\ell \phi_{1,x} \, GJ \, \phi_{1,x} \, dx}_{K_{11}} \, C_1 = \delta C_1 \underbrace{\int_0^\ell \left( \phi_1 \, m_t - \phi_{1,x} \, GJ \, \phi_{0,x} \right) dx}_{f_1}\]

where:

\[K_{11} = GJ \, \ell^3 / 3 \qquad f_1 = -\frac{m_t \ell^3}{6}\]

and so:

\[C_1 = -\frac{m_t}{2GJ}\]
Result: \[\theta = C_1 \phi_1 + \phi_0 = \frac{1}{2} \frac{m_t \ell^2}{GJ} \left( \frac{x}{\ell} - \frac{x^2}{\ell^2} \right) + \frac{\bar{\theta} x}{\ell}\]

As obtained previously (exact solution).

Exercise 3: Cantilever Beam on Elastic Foundation

Solution

Essential Conditions

\[\begin{cases} W(0) = 0 \\ W_{,x}(0) = 0 \end{cases} \quad \Rightarrow \quad \begin{aligned} \phi_i(0) &= 0 \\ \phi_{i,x}(0) &= 0 \end{aligned}\]

Trial Functions

The trial functions are taken as:

\[\phi_i = x^{i+1} \qquad i = 1, \ldots, N\]

Considering one single term:

\[\phi_1 = x^2\]

Principle of Virtual Work

The PVW reads:

\[\int_0^\ell \delta W_{,xx} \, EJ \, W_{,xx} \, dx + \int_0^{\ell/2} \delta W \, k \, W \, dx + \delta W(\ell) \, K \, W(\ell) = \delta W(\ell) \, P\]

Upon substitution of the Ritz expansion it is obtained:

\[\delta C_1 \left[ \int_0^\ell \phi_{1,xx} \, EJ \, \phi_{1,xx} \, dx + \int_0^{\ell/2} \phi_1 \, k \, \phi_1 \, dx + \phi_1(\ell) \, K \, \phi_1(\ell) \right] C_1 = \delta C_1 \, \phi_1(\ell) \, P\]

which leads to:

\[\left( 4EJ\ell + \frac{k\ell^5}{160} + \ell^4 K \right) C_1 = P\ell^2\]
Final Answer: \[C_1 = \frac{160 \, P\ell}{640 \, EJ + k\ell^4 + 160 \, \ell^3 K}\]

Exercise 4: First Bending Frequency — Variable Stiffness Beam

Determine the first bending natural frequency using 1 trigonometric function and the method of Ritz.

Given:

  • \(EJ = EJ_0 \sin \frac{\pi x}{\ell}\)

Solution

Ritz Solution — 1 Term

The essential conditions are:

\[W(0) = 0 \qquad W(\ell) = 0\]

The Ritz approximation is then taken as:

\[W = C_1 \sin \frac{\pi x}{\ell}\]

External Work for Free Vibrations

To evaluate the free vibrations, the external work is written as:

\[\delta W_e = -\int_V \left( \delta u \, \rho \ddot{u} + \delta w \, \rho \ddot{w} \right) dV\]

After expanding and simplifying (neglecting rotary inertia \(I_2\) and noting \(I_1 = 0\)):

\[\delta W_e = -\int_0^\ell \left( \delta u_0 \, I_0 \, \ddot{u}_0 + \delta w_0 \, I_0 \, \ddot{w}_0 \right) dx\]

The first contribution is associated with axial vibrations. The second one contributes to bending vibrations.

Eigenvalue Problem

Setting \(\delta W_i = \delta W_e\):

\[\int_0^\ell \delta w_{0,xx} \, EJ \, w_{0,xx} \, dx + \int_0^\ell \delta w_0 \, I_0 \, \ddot{w}_0 \, dx = 0\]

With \(w(x,t) = C_1 e^{i\omega t} \sin \frac{\pi x}{\ell} = C_1 \phi_1 e^{i\omega t}\):

\[\left[ \delta C_1 \underbrace{\int_0^\ell \phi_{1,xx} \, EJ \, \phi_{1,xx} \, dx}_{K_{11}} \, C_1 - \delta C_1 \, \omega^2 \underbrace{\int_0^\ell \phi_1 \, I_0 \, \phi_1 \, dx}_{M_{11}} \, C_1 \right] e^{i\omega t} = 0\]

The problem is then in the form:

\[(K_{11} - \omega^2 M_{11}) C_1 = 0\]

where:

\[K_{11} = \frac{4\pi^3 \, EJ_0}{3\ell^3} \qquad M_{11} = I_0 \frac{\ell}{2}\]
First Circular Frequency: \[\omega_1 = \sqrt{\frac{K_{11}}{M_{11}}} = \sqrt{\frac{EJ_0}{I_0} \frac{8\pi^3}{3\ell^4}}\]

Exercise 5: Natural Frequency of Cantilever with Tip Mass

Evaluate the first natural frequency considering a polynomial expansion. Elaborate the possibility of considering one-dof approximations using static deflections.

Solution

Weak Form

\[\delta W_i = \int_0^\ell \delta W_{,xx} \, EJ \, W_{,xx} \, dx\] \[\delta W_e = -\int_0^\ell \delta W \, I_0 \, \ddot{W} \, dx - \delta W(\ell) \, M \ddot{W}(\ell)\]

Ritz Functions — 1 Term (Polynomial)

The simplest choice is to consider a polynomial expansion:

\[W = C_1 x^2 \quad \Rightarrow \quad \phi_1 = x^2\]

(The constant and linear contributions are not accounted for as they do not satisfy the boundary conditions at x = 0)

Essential conditions:

\[\begin{cases} W(0) = 0 \\ W_{,x}(0) = 0 \end{cases}\]

Upon substitution in the weak form:

\[\delta C_1 \, K_{11} \, C_1 = -\delta C_1 \, M_{11} \, \ddot{C}_1 \quad \forall \, \delta C_1\]

where:

\[K_{11} = \int_0^\ell \phi_{1,xx} \, EJ \, \phi_{1,xx} \, dx = 4EJ\ell\] \[M_{11} = \int_0^\ell \phi_1 \, I_0 \, \phi_1 \, dx + \phi_1(\ell) \, M \, \phi_1(\ell) = \ell^4 \left( \frac{1}{5} I_0 \ell + M \right)\]

The equation of motion is:

\[M_{11} \ddot{C}_1 + K_{11} C_1 = 0\]

Setting \(C_1 = \hat{C}_1 \, e^{i\omega t}\):

\[\left( -\omega^2 M_{11} + K_{11} \right) C_1 = 0 \quad \Rightarrow \quad \omega^2 = \frac{K_{11}}{M_{11}}\]

Ritz Functions — 2 Terms

Expand to the 3rd order:

\[W = C_1 x^2 + C_2 x^3 \quad \Rightarrow \quad \phi_1 = x^2 \; ; \; \phi_2 = x^3\]

Additional stiffness and mass terms:

\[K_{12} = \int_0^\ell \phi_{1,xx} \, EJ \, \phi_{2,xx} \, dx = 6EJ\ell^2\] \[M_{12} = \int_0^\ell \phi_1 \, I_0 \, \phi_2 \, dx + \phi_1(\ell) \, M \, \phi_2(\ell) = \ell^5 \left( \frac{1}{6} I_0 \ell + M \right)\] \[K_{22} = \int_0^\ell \phi_{2,xx} \, EJ \, \phi_{2,xx} \, dx = 12 \, EJ \, \ell^3\] \[M_{22} = \int_0^\ell \phi_2 \, I_0 \, \phi_2 \, dx + \phi_2(\ell) \, M \, \phi_2(\ell) = \ell^6 \left( \frac{1}{7} I_0 \ell + M \right)\]

Ritz Functions — Static Deflections

The accuracy to dof ratio can be improved if the shape functions are properly selected. A good approach for obtaining a good solution using one single dof consists in selecting a static deflected shape.

Two cases:

  • Cantilever with tip load → appropriate when \(M >> I_0 \ell\)
  • Cantilever with uniform load → appropriate when \(I_0 \ell >> M\)

Case 1: \(M >> I_0 \ell\) — Tip Load Static Deflection

The deflected shape is selected by solving:

\[\begin{cases} EJ \, W_{,xxxx} = 0 \\ W(0) = 0 \\ W_{,x}(0) = 0 \end{cases} \text{(essential)} \qquad \begin{cases} W_{,xx}(\ell) = 0 \\ W_{,xxx}(\ell) = -\frac{P}{EJ} \end{cases} \text{(natural)}\]

The solution is:

\[W = \frac{P}{6EJ} \left( 3x^2 \ell - x^3 \right)\]

The Ritz function is then:

\[W = C_1 \phi_1 \quad \text{with} \quad \phi_1 = 3x^2 \ell - x^3\]

Note that the static analysis allows to obtain one single dof expansion where both quadratic and cubic terms are accounted for. Their relative weight is given by the solution of the static problem.

The stiffness and mass matrix are:

\[K_{11} = \int_0^\ell \phi_{1,xx} \, EJ \, \phi_{1,xx} \, dx = 12 \, EJ \, \ell^3\] \[M_{11} = \int_0^\ell \phi_1 \, I_0 \, \phi_1 \, dx + \phi_1(\ell) \, M \, \phi_1(\ell) = \ell^6 \left( \frac{33}{35} I_0 \ell + 4M \right)\]

and so: \(\omega^2 = K_{11}/M_{11}\)

Case 2: \(I_0 \ell >> M\) — Uniform Load Static Deflection

The deflected pattern is found by solving:

\[\begin{cases} EJ \, W_{,xxxx} = q \\ W(0) = 0 \\ W_{,x}(0) = 0 \end{cases} \text{(essential)} \qquad \begin{cases} W_{,xx}(0) = 0 \\ W_{,xxx}(0) = 0 \end{cases} \text{(natural)}\]

The problem is solved as \(W = W^H + W^P\):

\[W^P = \frac{1}{24} \frac{q}{EJ} x^4 \qquad W^H = A_0 + A_1 x + A_2 x + A_3 x^3\]

Imposing the boundary conditions:

\[W = \frac{1}{24} \frac{q}{EJ} \left( x^4 - 4x^3 \ell + 6x^2 \ell^2 \right)\]

The single-dof Ritz representation is:

\[W = C_1 \phi_1 \quad \text{with} \quad \phi_1 = x^4 - 4x^3 \ell + 6x^2 \ell\]

The stiffness and mass contributions are:

\[K_{11} = \frac{144 \, EJ \, \ell^5}{5}\] \[M_{11} = \ell^8 \left( \frac{104}{45} I_0 \ell + 9M \right)\]

and so: \(\omega_1 = K_{11} / M_{11}\)

Exercise 6: Two-Span Continuous Beam

Determine the first natural frequency by using a single-dof approximation.

Solution

Weak Form

Introduce two reference systems as illustrated in the figure (x₁ for left span, x₂ for right span):

\[\delta W_i = \int_0^\ell \delta W^1_{,xx} \, EJ \, W^1_{,xx} \, dx_1 + \int_0^{\alpha\ell} \delta W^2_{,xx} \, EJ \, W^2_{,xx} \, dx_2\] \[\delta W_e = -\int_0^\ell \delta W^1 \, I_0 \, \ddot{W}^1 \, dx_1 - \int_0^{\alpha\ell} \delta W^2 \, I_0 \, \ddot{W}^2 \, dx_2\]

Essential Conditions

The essential conditions of the problem are:

  • \(W^1(0) = 0\) — Left support
  • \(W^1(\ell) = W^2(0) = 0\) — Middle support (displacement)
  • \(W^1_{,x}(\ell) = W^2_{,x}(0)\) — Middle support (rotation continuity)
  • \(W^2(\alpha\ell) = 0\) — Right support

Ritz Functions

A possible choice is given by sine-type functions:

\[W^1 = C_1 \phi_1 \quad \text{with} \quad \phi_1 = \sin \frac{\pi x_1}{\ell}\] \[W^2 = C_2 \phi_2 \quad \text{with} \quad \phi_2 = \sin \frac{\pi x_2}{\alpha \ell}\]

The two functions clearly satisfy the essential conditions at the outer supports:

\[W^1(0) = C_1 \sin(0) = 0 \quad \checkmark\] \[W^2(\alpha\ell) = C_2 \sin \pi = 0 \quad \checkmark\]

The inner condition on transverse deflection is also satisfied:

\[W^1(\ell) = W^2(0) = 0 \quad \checkmark\]

The essential condition on rotation must be imposed. Computing the derivatives:

\[W^1_{,x} = \frac{\pi}{\ell} \cos \frac{\pi x_1}{\ell} \, C_1\] \[W^2_{,x} = \frac{\pi}{\alpha\ell} \cos \frac{\pi x_2}{\alpha\ell} \, C_2\]

Imposing \(W^1_{,x}(\ell) = W^2_{,x}(0)\):

\[\frac{\pi}{\ell} \cos \pi \, C_1 = \frac{\pi}{\alpha\ell} \cos(0) \, C_2\] \[-\frac{\pi}{\ell} C_1 = \frac{\pi}{\alpha\ell} C_2\]

Therefore:

\[C_2 = -\alpha \, C_1\]

One-DOF Representation

The one-dof representation of the deflected patterns is:

\[W^1 = C_1 \sin \frac{\pi x_1}{\ell} \qquad W^2 = -C_1 \alpha \sin \frac{\pi x_2}{\alpha\ell}\]

or equivalently:

\[\phi_1 = \sin \frac{\pi x_1}{\ell} \qquad \phi_2 = -\alpha \sin \frac{\pi x_2}{\alpha\ell}\]

where \(C_1\) is the single degree of freedom.

Stiffness and Mass Matrices

The stiffness matrix is obtained as:

\[K_{11} = \int_0^\ell \phi_{1,xx} \, EJ \, \phi_{1,xx} \, dx_1 + \int_0^{\alpha\ell} \phi_{2,xx} \, EJ \, \phi_{2,xx} \, dx_2 = \frac{EJ \, \pi^4 (\alpha + 1)}{2\alpha\ell^3}\]

The mass matrix reads:

\[M_{11} = \int_0^\ell \phi_1 \, I_0 \, \phi_1 \, dx_1 + \int_0^{\alpha\ell} \phi_2 \, I_0 \, \phi_2 \, dx_2 = \frac{1}{2} I_0 \ell \left( 1 + \alpha^3 \right)\]
First Natural Frequency: \[\omega_1^2 = \frac{K_{11}}{M_{11}} = \frac{EJ\pi^4}{(\alpha^2 - \alpha + 1) I_0 \ell^4 \alpha}\]

When \(\alpha = 1\) (two beams of equal length):

\[\omega_1^2 = \frac{EJ\pi^4}{I_0 \ell^4}\]

Corresponding to the value obtained for one single simply-supported beam.
