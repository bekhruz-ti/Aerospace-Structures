Trial Functions – Overview

The out-of-plane displacement is expanded by separation of variables as:

\[w = \sum_{ij}^{RS} c_{ij} \phi_i(x) \psi_j(y)\]

Trial functions (along x direction; y-direction analogous by replacing x with y and a with b)

Clamped-Clamped (CC)

\[\phi_i = \left(\frac{x}{a}\right)^{i+3} - 2\left(\frac{x}{a}\right)^{i+2} + \left(\frac{x}{a}\right)^{i+i} \quad \text{or} \quad \phi_i = 1 - \cos\frac{2i\pi x}{a}\]

Clamped-Free (CF)

\[\phi_i = \left(\frac{x}{a}\right)^{i+1}\]

Clamped-Supported (CS)

\[\phi_i = \left(1 - \frac{x}{a}\right)\left(\frac{x}{a}\right)^{i+1}\]

Supported-Supported (SS)

\[\phi_i = \left(1 - \frac{x}{a}\right)\left(\frac{x}{a}\right)^{i} \quad \text{or} \quad \phi_i = \sin\frac{i\pi x}{a}\]
Trial Functions – Clamped-Clamped (CC)
\[\phi_i = \left(\frac{x}{a}\right)^{i+3} - 2\left(\frac{x}{a}\right)^{i+2} + \left(\frac{x}{a}\right)^{i+i} \quad \text{or} \quad \phi_i = 1 - \cos\frac{2i\pi x}{a}\]
Left end:
\(\phi = 0\)
\(\phi_{,x} = 0\)
Right end:
\(\phi = 0\)
\(\phi_{,x} = 0\)
Trial Functions – Clamped-Free (CF)
\[\phi_i = \left(\frac{x}{a}\right)^{i+1}\]
Left end (clamped):
\(\phi = 0\)
\(\phi_{,x} = 0\)
Right end (free):
No displacement constraints
Trial Functions – Clamped-Supported (CS)
\[\phi_i = \left(1 - \frac{x}{a}\right)\left(\frac{x}{a}\right)^{i+1}\]
Left end (clamped):
\(\phi = 0\)
\(\phi_{,x} = 0\)
Right end (supported):
\(\phi = 0\)
Trial Functions – Supported-Supported (SS)
\[\phi_i = \left(\frac{x}{a} - 1\right)\left(\frac{x}{a}\right)^{i} \quad \text{or} \quad \phi_i = \sin\frac{i\pi x}{a}\]
Left end:
\(\phi = 0\)
Right end:
\(\phi = 0\)
Equation Reminder: Stiffness Matrix

Stiffness Matrix

\[\mathbf{K}_{RS \times RS} = D_{11}\left(\mathbf{I}_{xx}^{22} \otimes \mathbf{I}_{yy}^{00}\right) + D_{12}\left(\mathbf{I}_{xx}^{20} \otimes \mathbf{I}_{yy}^{02} + \mathbf{I}_{xx}^{02} \otimes \mathbf{I}_{yy}^{20}\right) + D_{22}\left(\mathbf{I}_{xx}^{00} \otimes \mathbf{I}_{yy}^{22}\right) + 4D_{66}\left(\mathbf{I}_{xx}^{11} \otimes \mathbf{I}_{yy}^{11}\right)\]

Where:

\[\left(\mathbf{I}_{xx}^{\alpha\beta}\right)_{ij} = \int_0^a \phi_i^{(\alpha)} \phi_j^{(\beta)} \mathrm{d}x \quad \text{and:} \quad \phi_i^{(\alpha)} = \frac{\mathrm{d}^\alpha \phi_i}{\mathrm{d}x^\alpha} \quad \text{with } i, j = 1 \ldots R\]
\[\left(\mathbf{I}_{yy}^{\alpha\beta}\right)_{ij} = \int_0^b \psi_i^{(\alpha)} \psi_j^{(\beta)} \mathrm{d}y \quad \text{and:} \quad \psi_i^{(\alpha)} = \frac{\mathrm{d}^\alpha \psi_i}{\mathrm{d}y^\alpha} \quad \text{with } i, j = 1 \ldots S\]
Equation Reminder: Load Vector

Load Vector

\[\mathbf{f}_{RS \times 1} = p \, \mathbf{J}_x \otimes \mathbf{J}_y\]

(dimensions: \(R \times 1\) and \(S \times 1\))

For uniform pressure:

\[\left(\mathbf{J}_x\right)_i = \int_0^a \phi_i \mathrm{d}x \quad \text{with} \quad i = 1 \ldots R\] \[\left(\mathbf{J}_y\right)_i = \int_0^b \psi_i \mathrm{d}y \quad \text{with} \quad i = 1 \ldots S\]

For concentrated load:

\[\left(\mathbf{J}_x\right)_i = \phi_i(x_0) \quad \text{with} \quad i = 1 \ldots R\] \[\left(\mathbf{J}_y\right)_i = \psi_i(y_0) \quad \text{with} \quad i = 1 \ldots S\]
Overview of the Program – Main

The structure of the program consists of following steps:

% --- Input data INPUT = input_model; [ PLATE, RITZ, LOAD ] = set_model( INPUT ); TRIAL = set_trial_fcs( PLATE.bcs ); % --- Evaluate integrals IRITZ = evaluate_integrals( TRIAL, PLATE, RITZ, LOAD ); % --- Assembly stiffness matrices K = assembly_stiffness( PLATE, IRITZ ); F = assembly_load_vector( LOAD, IRITZ ); % --- Solve the linear static problem RITZ.C = K \ F; % --- Post-process ... whatever you like
Overview of the Program – Structures (PLATE)

The functions used in the program are PLATE, RITZ, LOAD, TRIAL, IRITZ whose fields are:

PLATE Structure

PLATE = a: 400 a: plate dimension along x b: 200 b: plate dimension along y E: 72000 E: Young's modulus nu: 0.3000 nu: Poisson's ratio t: 1 t: thickness I0: 2.7000e-09 I0: plate inertia bcs: 'CCCC' bcs: boundary conditions D: [3×3 double] D: bending stiffness

Bending Stiffness Matrix:

\[\mathbf{D} = \frac{Et^3}{12(1-\nu^2)} \begin{bmatrix} 1 & \nu & 0 \\ \nu & 1 & 0 \\ 0 & 0 & \frac{1-\nu}{2} \end{bmatrix}\]
Overview of the Program – Structures (RITZ, LOAD)

RITZ Structure

RITZ = struct with fields: R: 3 R: number of trial functions along x S: 3 S: number of trial functions along y C: [9×1 double] C: Ritz amplitudes

LOAD Structure

LOAD = struct with fields: p0: 1 p0: pressure load intensity load_type: 'uniform' load_type: 'uniform' or 'concentrated' load_pos_x0: [] load_pos_x0: x-position of concentrated load (if any) load_pos_y0: [] load_pos_y0: y-position of concentrated load (if any)
Overview of the Program – Structures (TRIAL)

The trial functions can be defined as function handles. For instance, in the case of a simply-supported plate, using trigonometric functions, they are defined as:

TRIAL Structure

TRIAL = struct with fields: phi: @(x,a,i)sin(i*pi*x/a) phi: Ritz functions along x dphi: @(x,a,i)(i*pi/a)*cos(i*pi*x/a) dphi: x-derivative of phi ddphi: @(x,a,i)-(i*pi/a)^2*sin(i*pi*x/a) ddphi: x-derivative of dphi psi: @(x,a,i)sin(i*pi*x/a) psi: Ritz functions along y dpsi: @(x,a,i)(i*pi/a)*cos(i*pi*x/a) dpsi: x-derivative of psi ddpsi: @(x,a,i)-(i*pi/a)^2*sin(i*pi*x/a) ddpsi: x-derivative of dpsi
Overview of the Program – Structures (IRITZ)

The Ritz integrals are collected into the structure IRITZ whose fields are:

IRITZ Structure

IRITZ = struct with fields: Ixx_00: [3×3 double] Ixx_00: int_0^a phi*phi; Ixx_02: [3×3 double] Ixx_02: int_0^a phi*ddphi; Ixx_11: [3×3 double] ... and so on Ixx_20: [3×3 double] Ixx_22: [3×3 double] Iyy_00: [3×3 double] Iyy_02: [3×3 double] Iyy_11: [3×3 double] Iyy_20: [3×3 double] Iyy_22: [3×3 double] Jx: [3×1 double] Jx: Ritz integral along x of load vector Jy: [3×1 double] Jy: Ritz integral along y of load vector
Definition of the Trial Functions

In this function, the trial functions are defined as function handles. The choice of the functions along x and y depends on the boundary conditions to be accounted for:

function TRIAL = set_trial_fcs( bcs ) % ... select the trial functions depending on the boundary conditions TRIAL.phi = @(x,a,i) sin(i*pi*x/a); TRIAL.dphi = @(x,a,i) (i*pi/a) * cos(i*pi*x/a); TRIAL.ddphi = @(x,a,i) -(i*pi/a)^2* sin(i*pi*x/a); TRIAL.psi = @(x,a,i) sin(i*pi*x/a); TRIAL.dpsi = @(x,a,i) (i*pi/a) * cos(i*pi*x/a); TRIAL.ddpsi = @(x,a,i) -(i*pi/a)^2* sin(i*pi*x/a);

Example relative to trigonometric functions and SSSS boundary conditions

Evaluation of the Ritz Integrals

The Ritz integrals can be performed numerically using the integral.m function:

function IRITZ = evaluate_integrals( TRIAL, PLATE, RITZ, LOAD ) a = PLATE.a; b = PLATE.b; R = RITZ.R; S = RITZ.S; phi = TRIAL.phi; dphi = TRIAL.dphi; ddphi = TRIAL.ddphi; for i = 1 : R for j = 1 : S IRITZ.Ixx_00(i,j) = integral( @(x) phi(x,a,i) .* phi(x,a,j), 0, a ); % ... and so on end end
Assembly of the Stiffness Matrix

The assembly of the stiffness matrix is straightforward thanks to the use of the Kronecker product:

function K = assembly_stiffness( PLATE, IRITZ ) D11 = PLATE.D(1,1); D12 = PLATE.D(1,2); D22 = PLATE.D(2,2); D66 = PLATE.D(3,3); Ixx_22 = IRITZ.Ixx_22; Ixx_20 = IRITZ.Ixx_20; Ixx_02 = IRITZ.Ixx_02; Ixx_00 = IRITZ.Ixx_00; Ixx_11 = IRITZ.Ixx_11; Iyy_22 = IRITZ.Iyy_22; Iyy_20 = IRITZ.Iyy_20; Iyy_02 = IRITZ.Iyy_02; Iyy_00 = IRITZ.Iyy_00; Iyy_11 = IRITZ.Iyy_11; % --- Build stiffness matrix K = D11 * kron( Ixx_22, Iyy_00 ) + ... D12 * ( kron( Ixx_20, Iyy_02 ) + kron( Ixx_02, Iyy_20 ) ) + ... D22 * kron( Ixx_00, Iyy_22 ) + ... 4 * D66 * kron( Ixx_11, Iyy_11 );
