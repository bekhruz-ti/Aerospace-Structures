Problem 1: Cantilever Beam with Distributed Load

Find the minimum value \(\displaystyle\frac{EJ}{EJ_0}\) such that the vertical displacement at \(x = L/2\) is smaller than \(W_{max}\).

Given Data:
\(L = 1200\left(1 + \frac{B}{10}\right)\, \text{mm}\)
\(EJ_0 = 6 \cdot 10^7\, \text{N·mm}^2\)
\(q = 1.0 \cdot \left(1 + \frac{C}{10}\right)\, \text{N/mm}\)
\(W_{max} = 5.0\, \text{mm}\)

Solution (B = C = 0)

Step 1: Find the unknown reaction force \(X\)

Using the unit load method with a "Real" system (beam with load \(q\) and reaction \(X\)) and a "Dummy" system (beam with unit load 1):

The bending moment expressions are:

\[M_x = -Xz - \frac{qz^2}{2}\] \[\delta M_x = -z\]

Applying compatibility:

\[\int_0^L \frac{M_x \cdot \delta M_x}{EJ}\, dz = 0\]

Solving for \(X\):

\[X = -\frac{3}{8}qL\]

Step 2: Evaluate the displacement at \(x = L/2\)

For the dummy system with unit load at \(x = L/2\):

\[\delta M_x = -z + \frac{L}{2}\]

The displacement condition:

\[\int_{L/2}^L \frac{\delta M_x \cdot M_x}{EJ}\, dz = S = W_{max}\]

Solving for \(EJ\):

\[EJ = \frac{1}{W_{max}} \int_{L/2}^L \delta M_x \cdot M_x\, dz\]

Step 3: Calculate the ratio

\[\frac{EJ}{EJ_0} = \frac{1}{EJ_0} \cdot \frac{L^3(40X + 17qL)}{384\, W_{max}}\]

Final Answer: \(\displaystyle\frac{EJ}{EJ_0} = \frac{L^3(40X + 17qL)}{384\, EJ_0\, W_{max}}\)

Problem 2: Torsional Stiffness Ratio of Cross-Sections

Problem 3: Truss/Frame System with Spring

Determine the vertical displacement in correspondence of the spring. (refer to the \(x\) axis in the figure)

Given Data:
\(A = 250\, \text{mm}^2\)
\(a = 150\, \text{mm}\)
\(t = 0.6\left(1 + \frac{B}{10}\right)\, \text{mm}\)
\(L = 1200\left(1 + \frac{F}{10}\right)\, \text{mm}\)
\(K = 100\, \text{N/mm}\)
\(E = 72000\, \text{MPa}\)
\(\nu = 0.3\)
\(P = 1700\, \text{N}\)

Solution (B = F = 0)

Step 1: Evaluate section properties using symmetry

Shear:

\[A^* = 2at\] \[GA^* = 2atG\]

Torsional:

\[GJ = Ga^3t\]

Bending:

\[EJ = EAa^2\]

Step 2: Evaluate the unknown reaction force \(X\) at the spring

Using the unit load method with Real and Dummy systems:

Real System:
\(T = P + X\)
\(M = (P + X)z\)
\(M_t = (P - X)\frac{a}{2}\)
Dummy System:
\(\delta T = 1\)
\(\delta M = z\)
\(\delta M_t = -\frac{a}{2}\)

Step 3: Apply compatibility condition

\[\int_0^L \left(\frac{\delta T \cdot T}{GA^*} + \frac{\delta M \cdot M}{EJ} + \frac{\delta M_t \cdot M_t}{GJ}\right) dz + \frac{X}{k} = 0\]

Step 4: Solve for \(X\)

\[X = P \cdot \frac{\displaystyle\frac{a^2}{4GJ} - \frac{L^2}{3EJ} - \frac{1}{6A^*}}{\displaystyle\frac{1}{GA^*} + \frac{L^2}{3EJ} + \frac{a^2}{4GJ} + \frac{1}{kL}} = -222.55\, \text{N}\]

Step 5: Calculate displacement

\[S = \frac{X}{k}\]

Final Answer: \(S = \displaystyle\frac{X}{k} = -2.23\, \text{mm}\)

Problem 4: Cantilever Beam with Variable Stiffness - Ritz Method

Using the Ritz method and a 2-dof polynomial approximation, determine the strain energy.

Given Data:
\(L = 2700\, \text{mm}\)
\(EA_0 = 2.8 \cdot 10^5\, \text{N}\)
\(EA_1 = 6.5 \cdot 10^5\, \text{N}\)
\(k = 300\left(1 + \frac{B}{10}\right)\, \text{N/mm}\)
\(h = 2\left(1 + \frac{F}{10}\right)\, \text{N/mm}\)

Solution (B = F = 0)

Step 1: Define displacement approximation

\[u(x) = q_1 x + q_2 x^2\]

Step 2: Set up variational equation

\[\int_0^L \delta u_{,x}\, EA(x)\, u_{,x}\, dx + \delta u(L)\, k\, u(L) = \int_0^L \delta u\, h\, dx\]

where the axial stiffness varies linearly:

\[EA(x) = \alpha + \beta x\]

with:

\[\alpha = EA_0 \qquad \beta = \frac{EA_1 - EA_0}{L}\]

Step 3: Compute stiffness matrix entries

\[K_{11} = \alpha L + \frac{\beta L^2}{2} + kL^2\] \[K_{12} = K_{21} = \alpha L^2 + \frac{2}{3}\beta L^3 + kL^3\] \[K_{22} = \frac{4}{3}\alpha L^3 + \beta L^4 + kL^4\]

Step 4: Compute force vector entries

\[f_1 = \frac{L^2}{2}h\] \[f_2 = \frac{L^3}{3}h\]

Step 5: Solve system \(\underline{K}\,\underline{q} = \underline{f}\)

\[q_1 = 9.23 \cdot 10^{-3}\] \[q_2 = -2.51 \cdot 10^{-6}\]

Step 6: Calculate strain energy

\[U = \frac{1}{2}\underline{q}^T \underline{K}\, \underline{q}\]

Final Answer: \(U = 17151\, \text{N·mm}\)

Problem 5: Two-Segment Beam with Rigid Link - Ritz Method

Using the Ritz method and a 1-dof polynomial approximation, determine the strain energy.

Note: \(GJ\) varies linearly from \(GJ_0\) at \(x = 0\) to \(GJ_1\) at \(x = L_2\)

Given Data:
\(L_1 = 180\, \text{mm}\)
\(L_2 = 1800\, \text{mm}\)
\(GJ_0 = 1.1 \cdot 10^8\, \text{N·mm}^2\)
\(GJ_1 = 3.3 \cdot 10^8\, \text{N·mm}^2\)
\(P = 50\left(1 + \frac{D}{10}\right)\, \text{N}\)

Solution (D = 0)

Step 1: Define rotation approximation

\[\theta = q_1 x\]

Step 2: Define variable torsional stiffness

\[GJ(x) = \alpha + \beta x\]

where:

\[\alpha = GJ_0 \qquad \beta = \frac{GJ_1 - GJ_0}{L_2}\]

Step 3: Set up variational equation

\[\int_0^L \delta\theta'\, GJ(x)\, \theta'\, dx = \delta\theta(L_2)\, P\, L_1\]

This gives:

\[\left(\alpha L_2 + \frac{\beta L_2^2}{2}\right) q_1 = P\, L_1\, L_2\]

Step 4: Solve for \(q_1\)

\[q_1 = 4.091 \cdot 10^{-5}\]

Step 5: Calculate strain energy

\[U = \frac{1}{2}\left(\alpha L_2 + \frac{\beta L_2^2}{2}\right) q_1^2\]

Final Answer: \(U = 331.36\, \text{N·mm}\)

Problem 6: Truss Structure - Axial Stress in Stringer

Evaluate the axial stress in stringer #1 at \(x = 0\).

Given Data:
\(a = 180\, \text{mm}\)
\(A = 350\, \text{mm}^2\)
\(L = 1600\left(1 + \frac{A}{10}\right)\, \text{mm}\)
\(P = 1300\left(1 + \frac{F}{10}\right)\, \text{N}\)

Solution (A = F = 0)

Step 1: Calculate second moment of area

\[J_{xx} = Aa^2\]

Step 2: Determine bending moment at \(x = 0\)

\[M = PL\]

Step 3: Calculate axial stress in stringer #1

The distance from the neutral axis to stringer #1 is \(\displaystyle\frac{a\sqrt{2}}{2}\), so:

\[\sigma = \frac{M}{J_{xx}} \cdot \frac{a\sqrt{2}}{2}\]

Final Answer: \(\sigma = 23.35\, \text{MPa}\)
