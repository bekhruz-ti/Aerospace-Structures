Exercise 6

Determine the reaction force in the figure. The structure is loaded with a prescribed displacement \(\bar{u}\).

Data:
\(\ell = 1500\left(1 + \frac{E}{10}\right)\, \text{mm}\)
\(EJ = 10^{11}\)
\(\bar{u} = 10\, \text{mm}\)

Solution

Real System Analysis

Setting up the equilibrium equations with reaction forces \(X\) (at fixed support), \(Y\) (at roller), and applied force \(F\) (corresponding to prescribed displacement):

\[\begin{cases} X + Y = F \\ Y\ell - 2F\ell = 0 \end{cases}\]

Solving:

\[\Rightarrow \begin{cases} X = F - Y = -F \\ Y = 2F \end{cases}\]
Moment Expressions

For segment 1 (coordinate \(x_1\) from left):

\[M_1 = Fx_1\]

For segment 2 (coordinate \(x_2\) from middle):

\[M_2 = F\ell - Fx_2\]
Dummy System

Apply unit load at the prescribed displacement location.

Virtual moment expressions:

\[\delta M_1 = x_1\] \[\delta M_2 = \ell - x_2\]
Application of PCVW (Principle of Complementary Virtual Work)
\[\int_0^{\ell} \frac{\delta M_1 M_1}{EJ}\, dx_1 + \int_0^{\ell} \frac{\delta M_2 M_2}{EJ}\, dx_2 = \bar{u}\]
Solving for F
\[F = \frac{3EJ\bar{u}}{2\ell^3} = 444.44\, \text{N}\]

Final Answer: \(Y = 2F = 888.88\, \text{N}\)

Exercise 7

Determine the vertical displacement (positive in the downward direction) due to a torsional moment \(\hat{H}_t\).

Data:
\(\ell = 1200\, \text{mm}\)
\(EJ = 10^{10}\, \text{Nmm}^2\)
\(GJ = 10^8\, \text{Nmm}^2\)
\(\hat{H}_t = 10^5\left(1 + \frac{F}{10}\right)\, \text{Nmm}\)

Solution

Real System Analysis

Internal moment distributions:

Torsion in horizontal member:

\[M_{t_1} = \hat{H}_t\]

Bending moment in inclined member:

\[M_2 = \hat{H}_t\]
Dummy System

Apply unit vertical load at the corner. The virtual moment in the inclined member (coordinate \(x_2\)):

\[\delta M_2 = x_2\]
Application of PCVW
\[\int_0^{\ell} \frac{\delta M_2 \cdot M_2}{EJ}\, dx_2 = V\]

Final Answer: \(V = \frac{M_t}{EJ} \cdot \frac{\ell^2}{2} = 7.20\, \text{mm}\)

Exercise 8

Determine the vertical displacement \(V\).

Data:
\(\ell = 1200\, \text{mm}\)
\(EJ = 10^{12}\, \text{Nmm}^2\)
\(k = 750\left(1 + \frac{G}{10}\right)\, \text{N/mm}\)
\(q = 12\, \text{N/mm}\)

Solution

Real System Analysis

With reactions \(q\ell\) at both supports:

Moment expressions (with distributed load \(q\)):

\[M_1 = -q\ell x_1 + \frac{qx_1^2}{2}\] \[M_2 = -q\ell x_2 + \frac{qx_2^2}{2}\]
Dummy System

Apply unit load at midpoint with reactions \(\frac{1}{2}\) at each end:

\[\delta M_1 = -\frac{1}{2}x_1\] \[\delta M_2 = -\frac{1}{2}x_2\]
Application of PCVW
\[\int_0^{\ell} \frac{\delta M_1 M_1}{EJ}\, dx_1 + \int_0^{\ell} \frac{\delta M_2 M_2}{EJ}\, dx_2 + \frac{\delta F_s \cdot F_s}{k} = V\]

where:

\[\delta F_s = \frac{1}{2}\] \[F_s = q\ell\]

Final Answer: \(V = \frac{5}{24} \cdot \frac{q\ell^4}{EJ} + \frac{q\ell}{2k} = 14.78\, \text{mm}\)

Exercise 11

Determine the vertical displacement \(V\) in correspondence of the Tip of the rigid link in the figure.

The rigid body is located at \(x = \ell/2\), i.e. in the middle of the beam.

Use Ritz with polynomial expansion and the lowest possible number of dofs.

Data:
\(\ell = 1200\, \text{mm}\)
\(L = 200\, \text{mm}\)
\(EJ = 10^{11}\, \text{Nmm}^2\)
\(GJ_0 = 10^{10}\, \text{Nmm}^2\)
\(GJ_1 = 2 \cdot 10^{10}\, \text{Nmm}^2\)
\(\hat{\theta} = 10\, \text{deg}\)
\(F = 5000\left(1 + \frac{B}{10}\right)\)

Solution

Essential Conditions:
\(\theta(0) = 0\)   \(w(0) = 0\)
\(\theta(\ell) = \hat{\theta}\)   \(w'(0) = 0\)
1) Trial Function for Displacement \(w\)

Starting with:

\[w = a_0 + a_1\left(\frac{x}{\ell}\right) + a_2\left(\frac{x}{\ell}\right)^2\]

Imposing \(w(0) = 0\) and \(w'(0) = 0\), one obtains:

\[w = a_2\left(\frac{x}{\ell}\right)^2 \quad \text{so:} \quad w'' = a_2 \cdot \frac{2}{\ell^2}\]
2) Trial Function for Rotation \(\theta\)

The expansion can be represented as:

\[\theta = \theta_H + \theta_P\]

where \(\theta_P\) is the term restoring the non-homogeneous condition, and \(\theta_H\) is the expansion where non-homogeneous conditions are replaced by homogeneous ones.

Boundary conditions for \(\theta_H\):

\[\theta_H(0) = 0, \quad \theta_H(\ell) = 0\]

Setting \(\theta_H = b_0 + b_1\left(\frac{x}{\ell}\right) + b_2\left(\frac{x}{\ell}\right)^2\):

\[\theta_H(0) = b_0 = 0\] \[\theta_H(\ell) = b_1 + b_2 = 0 \Rightarrow b_1 = -b_2\]

Therefore:

\[\theta_H(\ell) = \left[\left(\frac{x}{\ell}\right)^2 - \left(\frac{x}{\ell}\right)\right]b_2\]

Particular solution:

\[\theta_P(x) = \frac{x}{\ell}\hat{\theta}\]

Complete solution:

\[\theta(x) = \left[\left(\frac{x}{\ell}\right)^2 - \left(\frac{x}{\ell}\right)\right]b_2 + \frac{x}{\ell}\hat{\theta}\] \[\theta' = \frac{1}{\ell}\left(\frac{2x}{\ell} - 1\right)b_2 + \frac{1}{\ell}\hat{\theta}\]
Application of PVW (Principle of Virtual Work)
\[\int_0^{\ell} \delta w'' EJ w''\, dx + \int_0^{\ell} \delta\theta' GJ \theta'\, dx = -\delta w(\ell)F\]

This leads to \(\underline{\underline{K}}\,\underline{u} = \underline{F}\), where:

\[\underline{\underline{K}} = \begin{bmatrix} \frac{4EJ}{\ell^3} & 0 \\ 0 & \frac{2GJ_0 + GJ_1}{6\ell} \end{bmatrix}\] \[\underline{F} = \begin{Bmatrix} -F \\ -\frac{GJ\hat{\theta}}{6\ell} \end{Bmatrix} \qquad \underline{u} = \begin{Bmatrix} a_2 \\ b_2 \end{Bmatrix}\]

Final Answer: \(V = -w(\ell/2) + \theta(\ell/2) \cdot L = 27.22\, \text{mm}\)

Exercise 12

Evaluate the rotation \(\theta\) at \(x = \ell/2\) solving the problem exactly (circular bar).

Report the result in deg.

Data:
\(\ell = 1500\, \text{mm}\)
\(GJ = 10^9\, \text{Nmm}^2\)
\(M_t = 200\left(1 + \frac{B}{10}\right)\, \text{N}\)

Solution

The governing equations are: \[\begin{cases} GJ\theta'' + M_t = 0 \\ \theta(0) = 0 \\ GJ\theta'(\ell) = 0 \end{cases}\]
General Solution

The solution of the system is found as:

\[\theta = \theta_H + \theta_P\]

a) Homogeneous solution:

\[GJ\theta_H'' = 0 \quad \Rightarrow \quad \theta_H' = a_0, \quad \theta_H = a_0 x + a_1\]

b) Particular solution:

\[GJ\theta_P'' = -M_t \quad \Rightarrow \quad \theta_P' = -\frac{M_t}{GJ}x, \quad \theta_P = -\frac{1}{2}\frac{M_t}{GJ}x^2\]

Complete solution:

\[\theta = a_1 + a_0 x - \frac{1}{2}\frac{M_t}{GJ}x^2\]
Applying Boundary Conditions

Imposing \(\theta(0) = 0\):

\[a_1 = 0\]

Imposing \(\theta'(\ell) = 0\):

\[a_0 = \frac{M_t \ell}{GJ}\]
Final Expression

Therefore:

\[\theta = \frac{M_t}{GJ}\left(\ell x - \frac{1}{2}x^2\right)\]

Final Answer: \(\theta(\ell/2) = \frac{M_t}{GJ} \cdot \frac{3\ell^2}{8} = 9.67\, \text{deg}\)

Exercise 24

Model the rib as a beam and determine the bending moment at \(x = \ell_1/2\).

Data:
\(\ell_1 = 500\, \text{mm}\)
\(\ell_2 = 1500\, \text{mm}\)
\(\ell_3 = 150\, \text{mm}\)
\(t = 1.0\, \text{mm}\)
\(A = 500\, \text{mm}^2\)
\(F = 6000\, \text{N}\)

Solution

Load Decomposition

The loading is decomposed into symmetric loading plus torsion (moment \(F\ell_1/2\)).

Shear Flow Determination

The shear flows are readily determined as:

  • Vertical webs: \(\frac{F}{2\ell_3}\)
  • Horizontal flows from torsion: \(\frac{F}{4\ell_3}\)

The shear flows for torsion were derived as:

\[\frac{F\ell_1}{2} = 2q\Omega_c \quad \Rightarrow \quad q = \frac{F\ell_1}{4\ell_1\ell_3} = \frac{F}{4\ell_3}\]
Superposition of Effects

By superposition of effects, combining the shear flows:

  • Left side: \(\frac{3F}{4\ell_3}\)
  • Right side: \(\frac{F}{4\ell_3}\)
  • Horizontal flows: \(\frac{F}{4\ell_3}\)

The equilibrating flows are equal and opposite.

Beam Model

The beam model of the rib has:

  • Force \(F\) up at left
  • Force \(\frac{3}{4}F\) down at left
  • Distributed load \(\frac{F}{4}\)
  • Moment \(M\) at right

Final Answer: \(M(x) = \frac{F}{4}x - \frac{F}{4}x = 0\)

– is quadratic

– involve shear and bending equilibrium

– is generally larger with respect to a corresponding (same dimensions and bending stiffness) compact section

– False

– True
