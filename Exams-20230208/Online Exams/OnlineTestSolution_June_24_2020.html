Problem 1: Beam Displacement with Triangular Load

Find the displacement at \(x = L/3\).

Data:
\(L = 2000\, \text{mm}\)
\(EA = 1.8 \cdot 10^6\, \text{N}\)
\(EJ = 1.8 \cdot 10^8\, \text{N mm}^2\)
\(n = 2(1 + F/10)\, \text{N/mm}\)

Solution \((F = 0)\)

Step 1: Evaluate the unknown reaction force \(X\)

Using the principle of virtual work with the real and dummy systems:

For the Real system:

\[N = X + \frac{nx^2}{2L}\]

For the Dummy system:

\[\delta N = 1\]

Applying the virtual work principle:

\[\int_0^L \frac{N \cdot \delta N}{EA}\, dx = 0 \quad \Rightarrow \quad X = -\frac{nL}{6}\]

Step 2: Evaluate the displacement

For the dummy system with unit load at \(x = 2L/3\):

\[\delta N = +1 \quad \text{for} \quad x \geq \frac{2}{3}L\]

Computing the displacement integral:

\[\int_{2L/3}^{L} \frac{N \cdot \delta N}{EA}\, dx = S\]

From which:

\[S = \frac{54X + 19Ln}{162\, EA} \cdot L\]

Final Answer: \(S = 0.274\, \text{mm}\)

Problem 2: Shear Stress in Closed Section Under Torsion

Determine the shear stress \(\tau\) in the panel of thickness \(t_2\).

Data:
\(A = 500\, \text{mm}^2\)      \(t_2 = 1.6\, \text{mm}\)
\(h = 200(1 + E/10)\, \text{mm}\)   \(M_t = 10^7\, \text{N mm}\)
\(b = 150\, \text{mm}\)
\(t_1 = 1.3\, \text{mm}\)

Solution \((E = 0)\)

Step 1: Calculate the shear flow

The shear flow in the panels is obtained from the Bredt formula:

\[2q\Omega = M_t\]

where the enclosed area is:

\[\Omega = bh + \frac{\pi h^2}{4}\]

Step 2: Calculate the shear stress

\[\tau = \frac{q}{t_2}\]

Final Answer: \(\tau = 50.88\, \text{MPa}\)

Problem 3: Shear Flow in Rectangular Frame

Calculate the shear flow \(q_2\).

Data:
\(A = 800(1 + A/10)\, \text{mm}^2\)   \(T = 7600\, \text{N}\)
\(a = 250\, \text{mm}\)
\(b = 600\, \text{mm}\)
\(t_1 = 1\, \text{mm}\)
\(t_2 = 2(1 + C/10)\, \text{mm}\)

Solution \((A = C = 0)\)

Step 1: Calculate section properties

\[J_{xx} = \frac{2}{2}Aa^2\] \[S'_{x_3} = \frac{Aa}{2}\] \[S'_{x_4} = \frac{3}{2}Aa\] \[S'_{x_1} = S'_{x_3}\]

Step 2: Apply the shear flow equation

\[q'_3 = -T \frac{S'_{x_3}}{J_{xx}} = -\frac{T}{3a}, \quad q'_1 = q'_3\] \[q'_4 = -T \frac{S'_{x_4}}{J_{xx}} = -\frac{T}{a}\]

Step 3: Apply the equivalence with respect to z

\[2\Omega_c q^* + 2q'_3 \Omega_3 + 2q'_4 \Omega_4 = 0\]

where:

\[\Omega_c = ab; \quad \Omega_3 = \Omega_4 = \frac{ab}{2}\]

So:

\[2q^* + q'_3 + q'_4 = 0\] \[\Rightarrow q^* = \frac{2}{3}\frac{T}{a} = 20.27\, \text{N/mm}\]

Final Answer: \(q_2 = q^* = 20.27\, \text{N/mm}\)

Problem 4: Ritz Method for Beam with Spring Support

Using the Ritz method and a 1-dof polynomial approximation, determine the vertical displacement in correspondence of the spring.

Data:
\(L = 750\, \text{mm}\)       \(q = 2\)
\(EJ = 1.25 \cdot 10^8\, \text{N mm}^2\)
\(k = 50(1 + A/10)\, \text{N/mm}\)

Solution \((A = 0)\)

Step 1: Assume displacement field

\[w = (x^3 - Lx^2)q_1\]

Step 2: Apply the virtual work principle

\[\int_0^L \delta w_{,xx}\, EJ\, w_{,xx}\, dx + \delta w(L/2)\, k\, w(L/2) = \int_0^L \delta w \cdot q \cdot \frac{x}{L}\, dx\]

From which:

\[\left(4EJL^3 + \frac{k}{64}L^6\right)q_1 = -\frac{1}{20}qL^4\]

Step 3: Calculate displacement at spring location

\[w(L/2) = -\frac{L^3}{8}q_1\]

Final Answer: \(w(L/2) = 4.768\, \text{mm}\)

Problem 5: L-Shaped Cantilever with Ritz Method

Use the Ritz method and the simplest polynomial approximation for evaluating \(S\).

Data:
\(L = 730\, \text{mm}\)
\(EA_1 = 1.15 \cdot 10^6(1 + F/10)\, \text{N}\)
\(EA_2 = 3 \cdot 10^5\, \text{N}\)
\(EJ = 1.8 \cdot 10^7\, \text{N mm}^2\)
\(P = 2500(1 + A/10)\, \text{N}\)

Solution \((A = F = 0)\)

Step 1: Problem decomposition

Due to uncoupling between axial and bending, we can consider only the axial response of beam 1 and the bending one of beam 2.

Step 2: Assume displacement fields

\[u_1 = q_1 x\] \[w_2 = q_2 x^2\]

Compatibility condition at junction:

\[u_1(L) = w_2(L) \quad \Rightarrow \quad q_1 = q_2 L\]

So:

\[u_1 = q_2 x_1 L\] \[w_2 = q_2 x_2^2\]

Step 3: Apply virtual work principle

\[\int_0^L \delta u_{1,x}\, EA_1\, u_{1,x}\, dx_1 + \int_0^L \delta w_{2,xx}\, EJ\, w_{2,xx}\, dx = \delta u_1(L)\, P_x\]

Which leads to:

\[(EA_1 L^3 + 4EJL)q_2 = P_x L^2\]

Step 4: Calculate displacement

\[S = w_2(L) = q_2 L^2\]

Final Answer: \(S = 1.12\, \text{mm}\)

Problem 6: Three-Dimensional Rib Structure Analysis

Model rib #2 as a beam, and evaluate the bending moment at \(x = a/2\) as \(M/M_{ref}\).

Data:
\(a = 700\, \text{mm}\)     \(P = 2000\, \text{N}\)
\(b = 200\, \text{mm}\)     \(q = 2(1 + F/10)\, \text{N/mm}\)
\(c = 100\, \text{mm}\)     \(M_{ref} = 9.8 \cdot 10^5\, \text{N·mm}\)
\(L = 2000\, \text{mm}\)
\(t = 0.6\, \text{mm}\)
\(A = 700\, \text{mm}^2\)

Solution

Step 1: Determine equilibrating shear flows on Rib #2

The equilibrating shear flows on Rib #2 are obtained with flow \(qa\) at top and \(qa/2b\) on the sides.

Step 2: Beam model analysis

With the beam model, the bending moment is:

\[M(x) = \frac{qa}{2} x - q\frac{x^2}{2}\]

Step 3: Evaluate moment at \(x = a/2\)

\[\frac{M(a/2)}{M_{ref}}\]

Final Answer: \(\displaystyle\frac{M(a/2)}{M_{ref}} = 0.125\)
