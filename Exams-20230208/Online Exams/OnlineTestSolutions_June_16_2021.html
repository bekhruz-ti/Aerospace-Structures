Online Test Solutions - June 16, 2021

Exercise 1

Determine the shear force \(T\) at the fixed end.

Data:
\(\ell_1 = 1000 \, \text{mm}\)   \(EA_1 = 1 \cdot 10^6 \, \text{N}\)   \(P = 1000 \, \text{N}\)
\(\ell_2 = 1000 \, \text{mm}\)   \(EA_2 = 3 \cdot 10^6 \, \text{N}\)
\(EJ_1 = 2 \cdot 10^{12} \, \text{Nmm}^2\)
\(EJ_2 = 1 \cdot 10^{12} \, \text{Nmm}^2\)
\(k = 2000\left(1 + \frac{A}{10}\right)\)

Solution

Real System

Beam 1:
\[N_1 = P\]
Beam 2:
\[N_2 = -V\] \[M_2 = -Px_2 - Wx_2\]

Dummy Systems

Dummy System #1 (unit load vertical):

Beam 1: \(\,^1\delta N_1 = 0\)

Beam 2: \(\,^1\delta N_2 = -1\), \(\,^1\delta M_2 = 0\)

Dummy System #2 (unit load horizontal):

Beam 1: \(\,^2\delta N_1 = 0\)

Beam 2: \(\,^2\delta N_2 = 0\), \(\,^2\delta M_2 = -x_2\)

Application of PVCW (Principle of Virtual Complementary Work):

Equation 1: \[\int_0^{\ell_2} \frac{N_2 \cdot \,^1\delta N_2}{EA_2} \, dx_2 + \frac{V}{k} = 0 \quad \Rightarrow \quad V = 0\]
Equation 2: \[\int_0^{\ell_2} \frac{M_2 \cdot \,^2\delta M_2}{EJ_2} \, dx_2 + \frac{W}{k} = 0\]

From which:

\[W = -\frac{Pk\ell_2^3}{k\ell_2^3 + 3EJ_2} = -400 \, \text{N}\]

And the shear force at the fixed end reads:

\(T = P + W = 600 \, \text{N}\)

Exercise 3

Determine the unknown reaction force \(X\) as reported in the sketch.

Data:
\(\ell = 1000\left(1 + \frac{C}{10}\right)\)   \(q = 0.1 \, \text{N/mm}\)
\(EA = 1 \cdot 10^6 \, \text{N}\)
\(EJ = 1 \cdot 10^9 \, \text{Nmm}^2\)

Solution

To simplify the solution, the symmetry of the problem can be exploited (clearly this is not strictly necessary).

Real System (half beam):
\[M = -W - \frac{qx^2}{2}\]
Dummy System:
\[\,^2\delta M = -1\]

Application of PCVW:

\[\int_0^{\ell/2} \frac{M \cdot \delta M}{EJ} \, dx = 0\]

From which:

\[W = -\frac{q\ell^2}{24}\]

And the reaction force \(X\) is then:

\[X = W + q\left(\frac{\ell}{2}\right)^2 \cdot \frac{1}{2} = \frac{1}{12}q\ell^2\]

\(X = 8333 \, \text{Nmm}\)

Exercise 10

Determine the vertical displacement \(v\).

Solve the problem using the Ritz method and consider the smallest possible number of dofs using a polynomial representation.

Data:
\(\ell = 1300\left(1 + \frac{A}{10}\right)\)   \(q = 100 \, \text{N/mm}\)
\(EA_1 = 3 \cdot 10^6 \, \text{N}\)   \(h = 70 \, \text{N/mm}\)
\(EJ_1 = 4 \cdot 10^{12} \, \text{Nmm}^2\)
\(EA_2 = 6 \cdot 10^6 \, \text{N}\)
\(EJ_2 = 8 \cdot 10^{12} \, \text{Nmm}^2\)

Solution

Compatibility relations:

\[d_2 = -c_1 \ell, \quad a_1 = -c_1\] \[u_1(\ell) = a_1, \quad w_2(\ell) = d_2\]

Trial Functions (10 dof):

\[u_1 = a_0 + a_1\left(\frac{x_1}{\ell}\right)\] \[w_1 = b_0 + b_1\left(\frac{x_1}{\ell}\right) + b_2\left(\frac{x_1}{\ell}\right)^2\] \[u_2 = c_0 + c_1\left(\frac{x_2}{\ell}\right)\] \[w_2 = d_0 + d_1\left(\frac{x_2}{\ell}\right) + d_2\left(\frac{x_2}{\ell}\right)^2\]

Essential Boundary Conditions (due to constraints) - 6 constraints:

At the fixed support:

\[u_1(0) = 0 \quad \Rightarrow \quad u_1 = a_1\left(\frac{x_1}{\ell}\right)\] \[w_1(0) = 0 \quad \Rightarrow \quad w_1 = b_2\left(\frac{x_1}{\ell}\right)^2\] \[w_1'(0) = 0\] \[u_2(0) = 0 \quad \Rightarrow \quad u_2 = c_1\left(\frac{x_2}{\ell}\right)\] \[w_2(0) = 0 \quad \Rightarrow \quad w_2 = d_2\left(\frac{x_2}{\ell}\right)^2\] \[w_2'(0) = 0\]

Essential Boundary Conditions at the Interface - 3 constraints:

\[u_1(\ell) = w_2(\ell) \quad \Rightarrow \quad a_1 = d_2\] \[w_1(\ell) = u_2(\ell) \quad \Rightarrow \quad b_2 = c_1\] \[w_1'(\ell) = -w_2'(\ell) \quad \Rightarrow \quad b_2 = -d_2\]

So the unknown amplitudes can be expressed as a function of one single amplitude, e.g. \(a_1\):

\[d_2 = a_1\] \[c_1 = -a_1\] \[b_2 = -a_1\]

Application of PVW:

Internal virtual work:

\[\delta W_i = \int_0^{\ell} \left(\delta u_1' EA_1 u_1' + \delta w_1'' EJ_1 w_1''\right) dx_1 + \int_0^{\ell} \left(\delta u_2' EA_2 u_2' + \delta w_2'' EJ_2 w_2''\right) dx_2\]

External virtual work:

\[\delta W_e = \int_0^{\ell} \delta u_1 \, h \, dx_1 + \int_0^{\ell} \delta w_2 \, q \, dx_2\]

From which:

\[K = \frac{EA_1}{\ell} + \frac{EA_2}{\ell} + \frac{4EJ_1}{\ell^3} + \frac{4EJ_2}{\ell^3}\] \[F = \frac{h\ell}{2} + \frac{q\ell}{3}\]

The linear static problem is then:

\[K a_1 = F\]

From which:

\[a_1 = 3.0876 \, \text{mm}\]

The vertical displacement \(v\) is: \(u_1(\ell) = a_1 = 3.0876 \, \text{mm}\)

Exercise (Torsion)

Compute the rotation of the loaded end.

Data:
\(\ell = 5000 \, \text{mm}\)   \(t = 0.8 \, \text{mm}\)
\(d_1 = 500\left(1 + \frac{F}{10}\right) \, \text{mm}\)
\(d_2 = 250\left(1 + \frac{E}{6}\right) \, \text{mm}\)
\(A = 500 \, \text{mm}^2\)
\(P = 8000 \, \text{N}\)
\(G = 27000 \, \text{MPa}\)

Solution

Geometric parameters:

\[\Omega = \frac{d_1 d_2}{2}\] \[d_3 = \sqrt{\left(\frac{d_1}{2}\right)^2 + \left(\frac{d_2}{2}\right)^2}\] \[d = \frac{d_2}{2} \cos 45°\]

The torsional constant of the section is evaluated referring to the Bredt's formula:

\[J = \frac{4\Omega^2}{\oint_p \frac{1}{t} \, dn} = \frac{4\Omega^2}{\frac{4d_3}{t}} = \frac{\Omega^2 t}{d_3}\]

But \(M_t = Pd = GJ\theta'\), so:

\[\theta' = \frac{Pd}{GJ}\]

And so: \(\theta = \theta' \ell = 0.6711 \, \text{deg}\)

Exercise 20

Compute the vertical displacement at \(x = \ell/2\) using Ritz and choosing between the approximations below:

\[w = C \cos\frac{\pi x}{\ell} \qquad w = Cx^2(x-\ell)^2\] \[w = Cx \qquad\qquad w = Cx(x-\ell)\] \[\qquad\qquad\qquad w = Cx^2(x-\ell)\]
Data:
\(\ell = 1500\left(1 + \frac{A}{10}\right) \, \text{mm}\)
\(EJ = 10^{10} \, \text{Nmm}^2\)
\(a = 1.0 \, \text{N/mm}\)

Solution

Consider the approximation:

\[w = Cx(x-\ell)\]

So:

\[w' = C \cdot 2x - C\ell = C(2x - \ell)\] \[w'' = 2C\]

The PVW reads:

\[\int_0^{\ell} \delta w'' \, EJ \, w'' \, dx = \int_0^{\ell} \delta w \cdot \frac{x}{\ell} \cdot a \, dx\]

And, upon substitution of the approximation:

\[\delta C \int_0^{\ell} 4EJ \, dx \cdot C = \delta C \int_0^{\ell} x(x-\ell) \cdot \frac{x}{\ell} \, dx\]

So:

\[(4EJ\ell) C = -\frac{a\ell^3}{12}\]

And then:

\[C = -\frac{a\ell^2}{48EJ}\]

The displacement at \(x = \ell/2\) is then:

\[w = Cx(x-\ell)\Big|_{x=\ell/2}\]

\(w = 2.64 \, \text{mm}\)

Exercise 23

Determine the axial stress \(\sigma_{zz}\) in the stringer #4 at \(z = \ell_3/2\).

Data:
\(\ell_1 = 400 \, \text{mm}\)   \(F = 8000 \, \text{N}\)
\(\ell_2 = 250\left(1 + \frac{B}{10}\right) \, \text{mm}\)
\(\ell_3 = 2000 \, \text{mm}\)
\(A = 500 \, \text{mm}^2\)

Solution

Second moment of area:

\[J_{xx} = 4A\left(\frac{\ell_2}{2}\right)^2 = A\ell_2^2\]

The bending moment reads:

\[M(z) = -Fz\]

So:

\[M\left(\frac{\ell_3}{2}\right) = -\frac{F\ell_3}{2}\]

From which:

\[\sigma_{zz} = -\frac{F\ell_3}{2} \cdot \left(-\frac{\ell_2}{2}\right) \cdot \frac{1}{J_{xx}} = \frac{F\ell_3}{4A\ell_2}\]

\(\sigma_{zz} = 32 \, \text{MPa}\)
