Question 1

Discuss and illustrate mathematically the relation between the Principle of Complementary Virtual Works (PCVW) and the compatibility conditions.

Solution

Principle of Complementary Virtual Works (PCVW):

The PCVW states that for a body in equilibrium, the complementary virtual work done by the actual stresses through virtual (compatible) strain variations equals the complementary virtual work done by the actual forces through virtual (compatible) displacement variations:

\[ \int_V \sigma_{ij} \delta\varepsilon_{ij}\, dV = \int_S t_i \delta u_i\, dS + \int_V f_i \delta u_i\, dV \]

Relation to Compatibility:

The PCVW is directly related to compatibility conditions because:

  1. The virtual displacements \(\delta u_i\) must be kinematically admissible (satisfy displacement boundary conditions)
  2. The virtual strains \(\delta\varepsilon_{ij}\) must be compatible with the virtual displacements through the strain-displacement relations: \[ \delta\varepsilon_{ij} = \frac{1}{2}\left(\frac{\partial \delta u_i}{\partial x_j} + \frac{\partial \delta u_j}{\partial x_i}\right) \]
  3. When the PCVW equation is satisfied for all admissible virtual displacement fields, it implies that the actual strain field is compatible

Application in Force Method:

In the force method (as used in Exercise 1), the PCVW is applied by:

  • Introducing redundant forces as unknowns
  • Writing compatibility equations ensuring that displacements at cut sections match
  • Using unit (dummy) load systems to compute flexibility coefficients

The resulting compatibility equations have the form:

\[ \delta_{ij} X_j = \Delta_i \]

where \(\delta_{ij}\) are flexibility coefficients computed using the PCVW, \(X_j\) are the redundant forces, and \(\Delta_i\) are the displacements due to external loads.
