Problem 1: Force-Based Approach #1

Determine the bending moment at \(x = \ell/2\).

Solve the problem using:

  1. a displacement-based approach
  2. a force-based approach

Assume that shearing deformation is negligible.

Data:
\(\ell = 1000\, \text{mm}\)
\(\hat{\theta} = 10\, \text{deg}\)
\(EJ = 10^{12}\, \text{Nmm}^2\)
Solution (Displacement-Based)
Governing Equations: \[EJ w_1^{IV} = 0 \qquad EJ w_2^{IV} = 0\]
Boundary Conditions:
\(w_1(0) = 0\) ... (1)
\(w_1'(0) = 0\) ... (2)
\(w_1(\ell) = 0\) ... (3)
\(w_2(0) = 0\) ... (4)
\(w_2'''(\ell) = 0\) ... (5)
\(w_2'(\ell) = -\hat{\theta}\) ... (6)

Continuity conditions:

\[w_1''(\ell) = w_2''(0) \quad \text{...(7)}\] \[w_1'(\ell) = w_2'(0) \quad \text{...(8)}\]
Solution of the ODEs: \[w_1 = A_0 + A_1 x_1 + A_2 x_1^2 + A_3 x_1^3\] \[w_2 = B_0 + B_1 x_2 + B_2 x_2^2 + B_3 x_2^3\]

Derivatives:

\[w_1' = A_1 + 2A_2 x_1 + 3A_3 x_1^2 \quad | \quad w_2' = B_1 + 2B_2 x_2 + 3B_3 x_2^2\] \[w_1'' = 2A_2 + 6A_3 x_1 \quad | \quad w_2'' = 2B_2 + 6B_3 x_2\] \[w_1''' = 6A_3 \quad | \quad w_2''' = 6B_3\]
Applying Boundary Conditions:

From (1)-(2): \(A_0 = A_1 = 0\)

From (3): \(A_2 + A_3 \ell = 0\)

From (4): \(B_0 = 0\)

From (5): \(B_3 = 0\)

From (6): \(B_1 + 2B_2 \ell = -\hat{\theta}\)

From (7): \(2A_2 + 6A_3 \ell = 2B_2\)

From (8): \(2A_2 \ell + 3A_3 \ell^2 = B_1\)

Solving the System: \[A_2 + A_3 \ell = 0 \quad \Rightarrow \quad A_2 = -A_3 \ell\] \[B_1 + 2B_2 \ell = -\hat{\theta} \quad \Rightarrow \quad B_1 = -\hat{\theta} - 2B_2 \ell\] \[2A_2 + 6A_3 \ell - 2B_2 = 0 \quad \Rightarrow \quad 2A_3 \ell = B_2\] \[2A_2 \ell + 3A_3 \ell^2 - B_1 = 0 \quad \Rightarrow \quad A_3 = -\frac{\hat{\theta}}{5\ell^2}\]
Final Coefficients:
\(A_0 = 0\)
\(A_1 = 0\)
\(A_2 = \frac{\hat{\theta}}{5\ell}\)
\(A_3 = -\frac{\hat{\theta}}{5\ell^2}\)
\(B_0 = 0\)
\(B_1 = -\frac{\hat{\theta}}{5}\)
\(B_2 = -\frac{2\hat{\theta}}{5\ell}\)
\(B_3 = 0\)
Bending Moment at \(x_2 = \ell/2\): \[M = -EJ w_2''(\ell/2)\] \[= -EJ(2B_2 + 3B_3 \ell) = EJ \cdot \frac{4}{5} \cdot \frac{\hat{\theta}}{\ell}\]

Final Answer: \(M = 1.3963 \times 10^8\, \text{Nmm}\)

Problem 2: Force-Based Approach #6

Use a force-based approach to determine the displacement \(S\). The contribution of shear deformability is negligible.

Data:
\(\ell = 1200\, \text{mm}\)
\(q = 2\, \text{N/mm}\)
\(EJ = 10^{12}\, \text{Nmm}^2\)
\(P = 2000\, \text{N}\)
Solution

As in the previous exercise, only the real forces producing a not null energy contribution are reported.

Moment Expressions:

Real system: \(M = -Px\)

Dummy system: \(\delta M = -x\)

Principle of Complementary Virtual Work (PCVW): \[\int_0^{2\ell} \delta M \cdot \frac{M}{EJ}\, dx = S\]
Evaluation: \[S = \frac{8}{3} \cdot \frac{P\ell^3}{EJ}\]

Final Answer: \(S = 9.22\, \text{mm}\)

Problem 3: Ritz Method #4

Determine the downward deflection of the point A, at the middle of the horizontal beam.

For this purpose, use the Ritz method where the deflection for beams #1 and #3 is taken as:

\[w_1 = C\,\phi(z_1) \quad \text{and} \quad w_3 = C\,\phi(z_3)\]

Note: \(C\) is the same owing to the symmetry of the problem.

Formulate the problem using the only dof \(C\) and choose between:

\[\phi(z) = \frac{z}{\ell}, \quad \phi(z) = \sin\frac{\pi z}{\ell}, \quad \phi(z) = \left(\frac{z}{\ell}\right)^2\]

Neglect the axial energy contributions.

Data:
\(\ell = 2000\, \text{mm}\)
\(EJ = 10^{12}\, \text{Nmm}^2\)
\(q = 50\, \text{N/mm}\)
Solution
Essential Boundary Conditions:
\(w_1(0) = 0\)
\(w_1'(0) = 0\)
\(w_3(0) = 0\)
\(w_3'(0) = 0\)

It follows that: \(\phi(z) = \left(\frac{z}{\ell}\right)^2\)

Approximation for Second Beam: \[w_2 = a_0 + a_1 \frac{z}{\ell} + a_2 \left(\frac{z}{\ell}\right)^2\]
Additional Essential Conditions: \[w_2(0) = 0\] \[w_2(\ell) = 0\] \[w_2'(0) = w_1'(\ell)\] \[w_2'(\ell) = -w_3'(\ell)\]
Solving for Coefficients:

From \(C \cdot \frac{z^2}{\ell^2}\) we get: \(C \cdot \frac{2z}{\ell^2}\)

\[a_0 = 0\] \[a_1 + a_2 = 0 \quad \Rightarrow \quad a_1 = 2C\] \[\frac{1}{\ell}a_1 = \frac{2}{\ell}C \quad \Rightarrow \quad a_2 = -2C\]

(Note: one equation is redundant due to the assumed symmetry of the solution)

Displacement Approximations: \[w_1 = C\left(\frac{x_1}{\ell}\right)^2\] \[w_3 = C\left(\frac{x_3}{\ell}\right)^2\] \[w_2 = C \cdot 2\left[\frac{x_2}{\ell} - \left(\frac{x_2}{\ell}\right)^2\right]\]
Principle of Virtual Work (PVW):

\(\delta W_i = \delta W_e\) with:

\[\delta W_i = \int_0^{\ell} \delta w_1'' EJ w_1'' dx_1 + \int_0^{\ell} \delta w_2'' EJ w_2'' dx_2 + \int_0^{\ell} \delta w_3'' EJ w_3'' dx_3\] \[\delta W_e = -\int_0^{\ell} \delta w_2 \cdot q \, dx_2\]
Substitution and Solution: \[\frac{24EJ}{\ell^3} \cdot C = -\frac{q\ell}{3}\] \[C = -\frac{q\ell^4}{72EJ}\]
Downward Deflection at Point A: \[w_A = -w_2(\ell/2) = -C \cdot 2\left[\frac{x_2}{\ell} - \left(\frac{x_2}{\ell}\right)^2\right]_{x_2=\ell/2}\] \[= -\frac{C}{2}\]

Final Answer: \(w_A = 5.56\, \text{mm}\)

Problem 4: Ritz Method #5

Determine the vertical displacement in the middle of the beam using the Ritz method. Approximate the displacement field using the simplest polynomial approximation.

Data:
\(\ell = 1500\, \text{mm}\)
\(P = 3000\, \text{N}\)
\(EJ_1 = 10^{10}\, \text{Nmm}^2\)
\(EJ_2 = 3 \times 10^{10}\, \text{Nmm}^2\)
Solution
Essential Boundary Conditions: \[w(0) = 0\] \[w'(0) = 0\] \[w(\ell) = 0\]
Polynomial Approximation: \[w = a_0 + a_1\left(\frac{x}{\ell}\right) + a_2\left(\frac{x}{\ell}\right)^2 + a_3\left(\frac{x}{\ell}\right)^3\]
Applying Boundary Conditions: \[a_0 = 0\] \[a_1 = 0\] \[a_2 + a_3 = 0\]

And so:

\[w = a_2\left[\left(\frac{x}{\ell}\right)^2 - \left(\frac{x}{\ell}\right)^3\right]\]
PVW: \(\delta W_i = \delta W_e\) \[\delta W_i = \int_0^{\ell/2} \delta w'' EJ_2 w'' dx + \int_{\ell/2}^{\ell} \delta w'' EJ_1 w'' dx\] \[\delta W_e = \delta w(\ell/2) \cdot P\]
Solving: \[\delta a_2 \cdot \frac{7EJ_1 + EJ_2}{2\ell^3} = \delta a_2 \cdot \frac{P}{8} \quad \forall\, \delta a_2\] \[\frac{7EJ_1 + EJ_2}{2\ell^3} \cdot a_2 = \frac{P}{8} \quad \Rightarrow \quad a_2 = 25.31\, \text{mm}\]
Displacement at Middle: \[w = a_2\left[\left(\frac{x}{\ell}\right)^2 - \left(\frac{x}{\ell}\right)^3\right]_{x=\ell/2}\] \[= \frac{a_2}{8}\]

Final Answer: \(w = 3.16\, \text{mm}\)

Problem 5: Semi-Monocoque #5

Determine the horizontal position of the shear center (report the coordinate \(x\)).

Data:
\(a = 400\, \text{mm}\)
\(t = 1.3\, \text{mm}\)
\(A = 800\, \text{mm}^2\)
Solution

The position of the \(x\) axis (principal system) is available from the symmetry of the section.

Section Properties: \[J_{xx} = 6A\left(\frac{a}{2}\right)^2 = \frac{3}{2}Aa^2\] \[S'_{x_1} = \frac{1}{2}Aa\] \[S'_{x_2} = \frac{3}{2}Aa\] \[S'_{x_3} = \frac{1}{2}Aa\]
Shear Flow Equations: \[q_1' = -T\frac{S'_{x_1}}{J_{xx}} = -\frac{1}{3}\frac{T}{a}\] \[q_2' = -T\frac{S'_{x_2}}{J_{xx}} = -\frac{T}{a}\] \[q_3' = -T\frac{S'_{x_3}}{J_{xx}} = -\frac{1}{3}\frac{T}{a}\]
Equivalence with Internal Moment (refer to stringer 3): \[2q_1'\Omega_1 + 2q^*\Omega_c = Tx\]

with: \(\Omega_1 = \frac{a^2}{2}\); \(\Omega_c = a^2\)

So:

\(-\frac{1}{3}Ta + 2q^*a^2 = Tx\)
Compatibility (\(\theta' = 0\)): \[q_1'a + q_2'\frac{a}{2} + q_3'a + q^*(2a + a) = 0\] \[-\frac{1}{3}T - \frac{1}{2}T - \frac{1}{3}T + 3a\,q^* = 0\] \[3a\,q^* = \frac{7}{6}T \quad \Rightarrow \quad q^* = \frac{7}{18}\frac{T}{a}\]
Substitution into Moment Equivalence: \[x = \frac{4}{9}a\]

Final Answer: \(x = 177.78\, \text{mm}\)

Problem 6: Semi-Monocoque #6

Evaluate the axial force carried by the top-right stringer at \(z = \ell/2\).

Data:
\(a = 250\, \text{mm}\)
\(b = 550\, \text{mm}\)
\(\ell = 3000\, \text{mm}\)
\(A = 400\, \text{mm}^2\)
\(P = 10\, \text{kN}\)
Solution
Shear Center Location: \[y_{sc} = \frac{6Aa}{8A} = \frac{3}{4}a\]
Second Moment of Area: \[J_{xx} = 6A\left(\frac{1}{4}a\right)^2 + 2A\left(\frac{3}{4}a\right)^2 = \frac{3}{2}Aa^2\]
Internal Actions: \[M(z) = Pz\]

and so:

\[M(\ell/2) = \frac{P\ell}{2}\]
Stress on Stringer 3: \[\sigma_{zz}(\ell/2) = \frac{M(\ell/2)}{J_{xx}} \cdot \frac{1}{4}a = \frac{P\ell}{12Aa} = 25\, \text{MPa}\]
Axial Force: \[N = \sigma_{zz} \cdot 3A = \frac{P\ell}{12a}\]

Final Answer: \(N = 30\, \text{kN}\)
