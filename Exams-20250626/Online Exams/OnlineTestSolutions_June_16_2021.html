Structural Mechanics Exam Solutions

Exercise 1

Determine the shear force \(T\) at the fixed end.

Given Data:

  • \(\ell_1 = 1000 \text{ mm}\)
  • \(\ell_2 = 1000 \text{ mm}\)
  • \(EA_1 = 1 \cdot 10^6 \text{ N}\)
  • \(EA_2 = 3 \cdot 10^6 \text{ N}\)
  • \(EJ_1 = 2 \cdot 10^{12} \text{ N mm}^2\)
  • \(EJ_2 = 1 \cdot 10^{12} \text{ N mm}^2\)
  • \(k = 2000\left(1 + A/_{10}\right)\)
  • \(\rho = 1000 \text{ N}\)

Solution

Step 1: Real System Analysis

Analyzing the real system with horizontal force \(W\) and vertical force \(V\) at the corner junction:

Beam 1 (horizontal):

\[N_1 = \rho\]

Beam 2 (vertical):

\[N_2 = -V\] \[M_2 = -\rho x_2 - W x_2\]

Step 2: Dummy System #1

Apply unit vertical force at corner junction:

Beam 1: \(^1\delta N_1 = 0\)

Beam 2: \(^1\delta N_2 = -1\), \(^1\delta M_2 = 0\)

Step 3: Dummy System #2

Apply unit horizontal force at corner junction:

Beam 1: \(^2\delta N_1 = 0\)

Beam 2: \(^2\delta N_2 = 0\), \(^2\delta M_2 = -x_2\)

Step 4: Application of PVCW

Equation 1:

\[\int_0^{\ell_2} \frac{N_2 \cdot ^1\delta N_2}{EA_2} dx_2 + \frac{V}{k} = 0 \quad \Rightarrow \quad V = 0\]

Equation 2:

\[\int_0^{\ell_2} \frac{M_2 \cdot ^2\delta M_2}{EJ_2} dx_2 + \frac{W}{k} = 0\]

From which:

\[W = -\frac{\rho k \ell_2^3}{k\ell_2^3 + 3EJ_2} = -400 \text{ N}\]

Step 5: Calculate Shear Force T

The shear force at the fixed end reads:

\[T = \rho + W = 1000 + (-400) = 600 \text{ N}\]
Final Answer: \[\boxed{T = 600 \text{ N}}\]

Exercise 3

Determine the unknown reaction force \(X\) as reported in the sketch.

Given Data:

  • \(\ell = 1000\left(1 + c/_{10}\right)\)
  • \(q = 0.1 \text{ N/mm}\)
  • \(EA = 1 \cdot 10^6 \text{ N}\)
  • \(EJ = 1 \cdot 10^9 \text{ N mm}^2\)

Solution

Step 1: Exploit Symmetry

To simplify the solution, the symmetry of the problem can be exploited (clearly this is not strictly necessary).

The beam is split into two symmetric half-beams, each of length \(\ell/2\).

Step 2: Real System

For the half-beam with moment \(W\) at the end:

\[M = -W - \frac{qx^2}{2}\]

Step 3: Dummy System

Apply unit moment at the end:

\[^1\delta M = -1\]

Step 4: Application of PCVW

By application of the Principle of Complementary Virtual Work:

\[\int_0^{\ell/2} \frac{M \cdot \delta M}{EJ} dx = 0\]

From which:

\[W = -\frac{q\ell^2}{24}\]

Step 5: Calculate Reaction Force X

The reaction force \(X\) is then:

\[X = W + q\left(\frac{\ell}{2}\right)^2 \cdot \frac{1}{2} = \frac{1}{12} q\ell^2 = 8333 \text{ N mm}\]
Final Answer: \[\boxed{X = \frac{1}{12} q\ell^2 = 8333 \text{ N·mm}}\]

Exercise 10

Determine the vertical displacement \(V\). Solve the problem using the Ritz method and consider the smallest possible number of dofs using a polynomial representation.

Given Data:

  • \(\ell = 1300\left(1 + A/_{10}\right)\)
  • \(q = 100 \text{ N/mm}\)
  • \(n = 70 \text{ N/mm}\)
  • \(EA_1 = 3 \cdot 10^6 \text{ N}\)
  • \(EJ_1 = 4 \cdot 10^{12} \text{ N mm}^2\)
  • \(EA_2 = 6 \cdot 10^6 \text{ N}\)
  • \(EJ_2 = 8 \cdot 10^{12} \text{ N mm}^2\)

Solution

Step 1: Define Trial Functions

The following set of trial functions is considered (10 dof initially):

\[u_1 = a_0 + a_1 \left(\frac{x_1}{\ell}\right)\] \[w_1 = b_0 + b_1\left(\frac{x_1}{\ell}\right) + b_2\left(\frac{x_1}{\ell}\right)^2\] \[u_2 = c_0 + c_1\left(\frac{x_2}{\ell}\right)\] \[w_2 = d_0 + d_1\left(\frac{x_2}{\ell}\right) + d_2\left(\frac{x_2}{\ell}\right)^2\]

Step 2: Apply Essential Boundary Conditions (due to constraints)

At base of member 1:

\[u_1(0) = 0 \quad \Rightarrow \quad u_1 = a_1\left(\frac{x_1}{\ell}\right)\] \[w_1(0) = 0, \quad w_1'(0) = 0 \quad \Rightarrow \quad w_1 = b_2\left(\frac{x_1}{\ell}\right)^2\]

At base of member 2:

\[u_2(0) = 0 \quad \Rightarrow \quad u_2 = c_1\left(\frac{x_2}{\ell}\right)\] \[w_2(0) = 0, \quad w_2'(0) = 0 \quad \Rightarrow \quad w_2 = d_2\left(\frac{x_2}{\ell}\right)^2\]

(6 constraints applied)

Step 3: Apply Essential Boundary Conditions at Interface

Continuity conditions at the junction:

\[u_1(\ell) = w_2(\ell) \quad \Rightarrow \quad a_1 = d_2\] \[w_1(\ell) = u_2(\ell) \quad \Rightarrow \quad b_2 = c_1\] \[w_1'(\ell) = -w_2'(\ell) \quad \Rightarrow \quad b_2 = -d_2\]

(3 additional constraints)

So the unknown amplitudes can be expressed as a function of one single amplitude, e.g. \(a_1\):

\[d_2 = a_1, \quad c_1 = -a_1, \quad b_2 = -a_1\]

Step 4: Apply PVW

Internal virtual work:

\[\delta W_i = \int_0^{\ell} \left(\delta u_1' EA_1 u_1' + \delta w_1'' EJ_1 w_1''\right) dx_1 + \int_0^{\ell} \left(\delta u_2' EA_2 u_2' + \delta w_2'' EJ_2 w_2''\right) dx_2\]

External virtual work:

\[\delta W_e = \int_0^{\ell} \delta u_1 \, n \, dx_1 + \int_0^{\ell} \delta w_2 \, q \, dx_2\]

Step 5: Solve the Linear System

From the virtual work equation:

\[K = \frac{EA_1}{\ell} + \frac{EA_2}{\ell} + \frac{4EJ_1}{\ell^3} + \frac{4EJ_2}{\ell^3}\] \[F = \frac{n\ell}{2} + \frac{q\ell}{3}\]

The linear static problem is:

\[K a_1 = F\]

From which:

\[a_1 = 3.0876 \text{ mm}\]
Final Answer: \[\boxed{V = u_1(\ell) = a_1 = 3.0876 \text{ mm}}\]

Exercise (Torsion)

Compute the rotation of the loaded end.

Given Data:

  • \(\ell = 5000 \text{ mm}\)
  • \(t = 0.8 \text{ mm}\)
  • \(d_1 = 500\left(1 + F/_{10}\right) \text{ mm}\)
  • \(d_2 = 250\left(1 + E/_6\right) \text{ mm}\)
  • \(A = 500 \text{ mm}^2\)
  • \(P = 8000 \text{ N}\)
  • \(G = 27000 \text{ MPa}\)

Solution

Step 1: Calculate Section Properties

Enclosed area:

\[\Omega = \frac{d_1 d_2}{2}\]

Diagonal length:

\[d_3 = \sqrt{\left(\frac{d_1}{2}\right)^2 + \left(\frac{d_2}{2}\right)^2}\]

Moment arm:

\[d = \frac{d_2}{2} \cos 45°\]

Step 2: Calculate Torsional Constant using Bredt's Formula

The torsional constant of the section is evaluated referring to Bredt's formula:

\[J = \frac{4\Omega^2}{\oint_p \frac{1}{t} dn} = \frac{4\Omega^2}{4d_3/t} = \frac{\Omega^2 t}{d_3}\]

Step 3: Calculate Rotation

The torsional moment is:

\[M_t = Pd = GJ\theta'\]

So the rate of twist:

\[\theta' = \frac{Pd}{GJ}\]

And the total rotation:

\[\theta = \theta' \ell\]
Final Answer: \[\boxed{\theta = 0.6711°}\]

Exercise 20

Compute the vertical displacement at \(x = \ell/2\) using Ritz and choosing between the approximations below:

Trial function options:

  • \(w = c \cos \frac{\pi x}{\ell}\)
  • \(w = c x^2 (x - \ell)^2\)
  • \(w = c x\)
  • \(w = c x (x - \ell)\)
  • \(w = c x^2 (x - \ell)\)

Given Data:

  • \(\ell = 1500 \left(1 + A/_{10}\right) \text{ mm}\)
  • \(EJ = 10^{10} \text{ N mm}^2\)
  • \(a = 1.0 \text{ N/mm}\)

Solution

Step 1: Choose Approximation

Consider the approximation:

\[w = c x (x - \ell)\]

So:

\[w' = c \cdot 2x - c\ell = c(2x - \ell)\] \[w'' = 2c\]

Step 2: Apply PVW

The Principle of Virtual Work reads:

\[\int_0^{\ell} \delta w'' \, EJ \, w'' dx = \int_0^{\ell} \delta w \, \frac{x}{\ell} \, a \, dx\]

Upon substitution of the approximation:

\[\delta c \int_0^{\ell} 4 EJ \, dx \cdot c = \delta c \int_0^{\ell} x(x - \ell) \frac{x}{\ell} \, dx\]

So:

\[(4 EJ \ell) c = -\frac{a\ell^3}{12}\]

Step 3: Solve for c

And then:

\[c = -\frac{a\ell^2}{48 EJ}\]

Step 4: Calculate Displacement

The displacement at \(x = \ell/2\) is then:

\[w = c \, x (x - \ell) \bigg|_{x = \ell/2} = c \cdot \frac{\ell}{2} \cdot \left(\frac{\ell}{2} - \ell\right) = c \cdot \frac{\ell}{2} \cdot \left(-\frac{\ell}{2}\right) = -\frac{c\ell^2}{4}\]
Final Answer: \[\boxed{w\left(\frac{\ell}{2}\right) = 2.64 \text{ mm}}\]

Exercise 23

Determine the axial stress \(\sigma_{zz}\) in the stringer #4 at \(z = \ell_3/2\).

Given Data:

  • \(\ell_1 = 400 \text{ mm}\)
  • \(\ell_2 = 250 \left(1 + B/_{10}\right) \text{ mm}\)
  • \(\ell_3 = 2000 \text{ mm}\)
  • \(A = 500 \text{ mm}^2\)
  • \(F = 8000 \text{ N}\)

Solution

Step 1: Calculate Second Moment of Area

The second moment of area about the x-axis:

\[J_{xx} = 4A \left(\frac{\ell_2}{2}\right)^2 = A\ell_2^2\]

Step 2: Calculate Bending Moment

The bending moment reads:

\[M(z) = -Fz\]

So at \(z = \ell_3/2\):

\[M\left(\frac{\ell_3}{2}\right) = -\frac{F\ell_3}{2}\]

Step 3: Calculate Axial Stress

Using the bending stress formula:

\[\sigma_{zz} = -\frac{M \cdot y}{J_{xx}}\]

For stringer #4 at \(y = -\ell_2/2\):

\[\sigma_{zz} = -\frac{F\ell_3}{2}\left(-\frac{\ell_2}{2}\right)\frac{1}{J_{xx}} = \frac{F\ell_3}{4A\ell_2}\]
Final Answer: \[\boxed{\sigma_{zz} = 32 \text{ MPa}}\]
