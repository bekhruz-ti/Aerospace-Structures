Displacement of Beams Systems II

Problem 1: EXAM 13/06/2023

Hyperstatic

Analyze the 3D L-shaped frame structure and determine the displacement \(u\) at point C.

Node Coordinates:

  • \(A: (0, 1000, 0)\, \text{mm}\)
  • \(B: (0, 1000, 1000)\, \text{mm}\)
  • \(C: (0, 0, 1000)\, \text{mm}\)
  • \(D: (1000, 0, 1000)\, \text{mm}\)

Given Data:

  • \(F = 1000\, \text{N}\)
  • \(EA = 10^6\, \text{N}\)
  • \(EI_{xx} = EI_{yy} = GJ = 10^{10}\, \text{N} \cdot \text{mm}^2\)
  • \(\ell = 1000\, \text{mm}\)

Solution

Strategy: If we are smart, we can avoid computing all the reaction forces by using the Principle of Complementary Virtual Work (PCVW) with appropriate virtual systems.

Virtual Systems Setup

Virtual 1: Apply unit force at point A in the \(z\)-direction (same direction as unknown reaction \(R_{zA}\)). This will be used to find the redundant reaction.

Virtual 2: Apply unit force at point C in the direction of desired displacement \(u\). This will be used to find the displacement once \(R_{zA}\) is known.

Internal Actions

Element 1 (Segment along \(z_1\))
Real
\[T_{z1} = -R_{zA}\]
Virtual 1
\[T_{z1}^1 = -1\]
Virtual 2

Unloaded

Element 2 (Segment along \(z_2\))
Real
\[\begin{cases} T_{y2} = -R_{zA} \\ M_{x2} = -R_{zA} \cdot z_2 \end{cases}\]
Virtual 1
\[\begin{cases} T_{y2}^1 = -1 \\ M_{x2}^1 = -z_2 \end{cases}\]
Virtual 2

Unloaded

Element 3 (Segment along \(z_3\), includes applied force F)
Real
\[\begin{cases} T_{y3} = -R_{zA} - F \\ M_{x3} = -(R_{zA} + F) \cdot z_3 \\ M_{z3} = -R_{zA} \cdot \ell \end{cases}\]
Virtual 1
\[\begin{cases} T_{y3}^1 = -1 \\ M_{x3}^1 = -z_3 \\ M_{z3}^1 = -\ell \end{cases}\]
Virtual 2
\[\begin{cases} T_{y3}^{11} = -1 \\ M_{x3}^{11} = -z_3 \\ M_{z3}^{11} = 0 \end{cases}\]

PCVW - Virtual 1 (Finding \(R_{zA}\))

External virtual work:

\[\delta W_e = 1 \cdot 0 = 0\]

Internal virtual work:

\[\delta W_i = \int_0^\ell T_{z1}^1 \cdot \frac{T_{z1}}{EA} \, dz_1 + \int_0^\ell M_{x2}^1 \cdot \frac{M_{x2}}{EI} \, dz_2 + \int_0^\ell \left( M_{x3}^1 \frac{M_{x3}}{EI} + M_{z3}^1 \frac{M_{z3}}{GJ} \right) dz_3\]

Substituting the internal actions:

\[\delta W_i = \frac{1}{EA} \int_0^\ell R_{zA} \, dz_1 + \frac{1}{EI} \int_0^\ell R_{zA} \cdot z_1^2 \, dz_1 + \frac{1}{EI} \int_0^\ell (R_{zA}+F) z_3^2 \, dz_3 + \frac{1}{GJ} \int_0^\ell R_{zA} \ell^2 \, dz_3\]

Evaluating the integrals:

\[\delta W_i = \frac{R_{zA} \ell}{EA} + \frac{R_{zA} \ell^3}{3EI} + \frac{(R_{zA}+F)\ell^3}{3EI} + \frac{R_{zA} \ell^3}{GJ}\]

Applying \(\delta W_e = \delta W_i\):

\[\boxed{R_{zA} = -125\, \text{N}}\]

PCVW - Virtual 2 (Finding displacement \(u\))

External virtual work:

\[\delta W_e = 1 \cdot u\]

Internal virtual work (only Element 3 contributes):

\[\delta W_i = \int_0^\ell M_{x3}^{11} \cdot \frac{M_{x3}}{EI} \, dz_3 = \frac{1}{EI} \int_0^\ell (R_{zA} + F) \cdot z_3^2 \, dz_3\]

Evaluating:

\[\delta W_i = \frac{1}{EI} \left[ \frac{1}{3} (R_{zA} + F) \cdot z_3^3 \right]_0^\ell = \frac{(R_{zA} + F)\ell^3}{3EI}\]

Substituting \(R_{zA} = -125\, \text{N}\), \(F = 1000\, \text{N}\), \(\ell = 1000\, \text{mm}\), \(EI = 10^{10}\, \text{N} \cdot \text{mm}^2\):

\[u = \frac{(-125 + 1000)(1000)^3}{3 \times 10^{10}} = \frac{875 \times 10^9}{3 \times 10^{10}}\]
Final Answer: \[\boxed{u = 29.1667\, \text{mm}}\]

Problem 2: EXAM 09/09/2024

Isostatic

Analyze the triangular truss structure and compute the horizontal displacement \(u\) at the loaded node.

Truss Properties:

  • → ALL TRUSS
  • → ONLY AXIAL LOADS
  • → NO MOMENTS, NO SHEARS
Note: Node 4 is free to translate. The bending and torsional stiffnesses given are a "BAIT" - they are not needed since truss members carry only axial loads!

Given Data:

  • \(\ell = 1000\, \text{mm}\)
  • \(EA = 10^8\, \text{N}\)
  • \(EI_{xx} = EI_{yy} = 10^{13}\, \text{N} \cdot \text{mm}^2\) (not needed for truss)
  • \(GJ = 7 \times 10^9\, \text{N} \cdot \text{mm}^2\) (not needed for truss)
  • \(F = 1000\, \text{N}\)
  • \(\theta = 30°\)

Solution

Internal Actions - Using Symmetry

Due to symmetry of the structure and loading, the vertical reactions at the supports are \(F/2\) each, and the horizontal reactions are \(1/2\) each.

Member 1 Analysis (Diagonal members)

Real Loading:

From equilibrium at the loaded node:

\[T_1 \cdot \cos\theta = F\] \[T_1 = \frac{F}{\cos\theta} = \frac{F}{\cos 30°} = \frac{F}{\frac{\sqrt{3}}{2}} = \frac{2F}{\sqrt{3}}\]

Virtual Loading:

Apply unit force in direction of desired displacement:

\[T_1^1 \cdot \cos\theta = \frac{1}{2}\] \[T_1^1 = \frac{1}{2\cos 30°} = \frac{1}{\sqrt{3}}\]

Member 2 Analysis (Vertical/side member)

Real Loading:

Horizontal equilibrium (trivial):

\[T_1 \cdot \cos\theta = \frac{F}{2}\]

Vertical equilibrium:

\[T_2 = T_1 \sin\theta = \frac{F}{\sqrt{3}} \cdot \frac{1}{2} = \frac{F}{2\sqrt{3}}\]

Virtual Loading:

\[T_2^1 = \frac{1}{2\sqrt{3}}\]

PCVW Application

External virtual work:

\[\delta W_e = 1 \cdot u\]

Internal virtual work (factor of 2 for symmetric diagonal members):

\[\delta W_i = 2 \cdot \frac{1}{EA} \int_0^\ell T_1^1 \, T_1 \, dz_1 + \frac{1}{EA} \int_0^\ell T_2^1 \, T_2 \, dz_2\]

Computing the products:

\[T_1^1 \cdot T_1 = \frac{1}{\sqrt{3}} \cdot \frac{F}{\sqrt{3}} = \frac{F}{3}\] \[T_2^1 \cdot T_2 = \frac{1}{2\sqrt{3}} \cdot \frac{F}{2\sqrt{3}} = \frac{F}{12}\]

Substituting:

\[\delta W_i = \frac{2}{EA} \int_0^\ell \frac{F}{3} \, dz_1 + \frac{1}{EA} \int_0^\ell \frac{F}{12} \, dz_2\] \[\delta W_i = \frac{2F\ell}{3EA} + \frac{F\ell}{12EA}\]

Applying \(\delta W_e = \delta W_i\):

\[u = \frac{2F\ell}{3EA} + \frac{F\ell}{12EA} = \frac{F\ell}{EA}\left(\frac{2}{3} + \frac{1}{12}\right) = \frac{F\ell}{EA} \cdot \frac{9}{12} = \frac{3F\ell}{4EA}\]

Substituting values: \(F = 1000\, \text{N}\), \(\ell = 1000\, \text{mm}\), \(EA = 10^8\, \text{N}\):

\[u = \frac{3 \times 1000 \times 1000}{4 \times 10^8} = \frac{3 \times 10^6}{4 \times 10^8} = 0.0075\, \text{mm}\]
Final Answer: \[\boxed{u = 0.0075\, \text{mm}}\]
